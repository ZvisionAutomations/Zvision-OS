# Task: Analyze Leverage & Wealth Strategy

## Metadata
- **Squad:** squad-council
- **Agent:** Naval Ravikant
- **Complexity:** Standard

## Objetivo
Aplicar o framework de criacao de riqueza de Naval a um negocio ou carreira. Identificar specific knowledge, escolher o tipo certo de leverage, e mapear oportunidades de compounding.

## Passos

### 1. Identify Specific Knowledge
- O que parece diversao para voce mas trabalho para outros?
- Que conhecimento unico voce tem que nao pode ser facilmente treinado?
- Onde sua curiosidade genuina aponta?
- O que voce faz naturalmente que outros acham difícil?

### 2. Map Leverage Options
| Tipo | Permissao | Escalabilidade | Atual? | Potencial? |
|------|-----------|----------------|--------|------------|
| Labor | Precisa | Linear | | |
| Capital | Precisa | Alta | | |
| Code | Nao precisa | Infinita | | |
| Media | Nao precisa | Infinita | | |

Perguntas-chave:
- Qual leverage voce esta usando hoje?
- Qual leverage voce DEVERIA estar usando?
- Como migrar para leverage permissionless (code/media)?

### 3. Apply Productize Yourself
Formula: Specific Knowledge x Leverage x Accountability = Wealth
- Como transformar seu specific knowledge em um produto?
- Que leverage amplifica melhor esse conhecimento?
- Voce esta tomando accountability (colocando seu nome)?

### 4. Identify Compounding Opportunities
- Conhecimento: que aprendizados compoem ao longo do tempo?
- Relacionamentos: quais relacoes de longo prazo cultivar?
- Reputacao: como construir track record que compoe?
- Habilidades: que skills adjacentes potencializam suas existentes?

### 5. Long-Term Game Assessment
- Esta jogando jogos de longo prazo com pessoas de longo prazo?
- Decisoes atuais compoem ou deterioram ao longo de decadas?
- O que voce faria diferente se o horizonte fosse 10 anos?
