## Confidence Levels

| Level | Criterio | Acao |
|-------|----------|------|
| **HIGH** | Match claro de keywords, dominio unico, intencao inequivoca | Rotear para primary advisor |
| **MEDIUM** | Multiplos dominios relevantes, leve ambiguidade | Sugerir primary + secondary, ou convocar painel |
| **LOW** | Sem match claro, request vago, cross-domain demais | Ficar com Zenith, fazer perguntas clarificadoras |
