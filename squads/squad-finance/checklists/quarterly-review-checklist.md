# Quarterly Financial Review Checklist

## 1. Revenue Analysis (Flow)
- [ ] Revenue waterfall (pipeline → bookings → revenue → billing → cash)
- [ ] Revenue breakdown por cliente, servico e tipo
- [ ] Concentracao de receita avaliada (HHI, top client %)
- [ ] MRR/ARR decomposicao (new, expansion, contraction, churn)
- [ ] NRR e GRR calculados e comparados com benchmark
- [ ] DSO e collection rate trimestrais
- [ ] Forecast de receita por cliente atualizado

## 2. Profitability Deep-Dive (Margin)
- [ ] P&L trimestral consolidado
- [ ] Gross margin e EBITDA trimestrais
- [ ] P&L por cliente (top 10)
- [ ] Client Profitability Matrix atualizada
- [ ] Unit economics atualizados (CAC, LTV, Magic Number)
- [ ] Utilizacao da equipe no trimestre
- [ ] Revenue per employee trimestral
- [ ] Benchmarks vs industria comparados

## 3. Budget Variance (Vault)
- [ ] Budget vs actual trimestral por rubrica
- [ ] Variancia calculada com root cause
- [ ] Classificacao: one-time vs recorrente vs estrutural
- [ ] Spending patterns identificados
- [ ] Acoes corretivas definidas para desvios significativos

## 4. Forecast Update (Vault)
- [ ] Rolling forecast atualizado com dados reais
- [ ] Forecast accuracy do trimestre calculada (target >=90%)
- [ ] Cash flow projetado para proximos 6-12 meses
- [ ] Runway atualizado
- [ ] 3 cenarios revisados (pessimista, base, otimista)
- [ ] EAC anual atualizado

## 5. Pricing Review (Mint)
- [ ] Rate realization trimestral
- [ ] Average deal size trend
- [ ] Win rate por faixa de preco
- [ ] Descontos concedidos vs politica
- [ ] Ajustes de preco recomendados (se necessario)

## 6. Board Report (Ledger)
- [ ] Executive summary (1 pagina, nao-tecnico)
- [ ] Revenue analysis (2-3 paginas)
- [ ] Profitability analysis (2-3 paginas)
- [ ] Budget performance (1-2 paginas)
- [ ] Updated forecast com 3 cenarios (1-2 paginas)
- [ ] Pricing recommendations (1 pagina)
- [ ] Risk register atualizado
- [ ] Top 5 acoes para proximo trimestre com responsavel e prazo
- [ ] Comparacao QoQ e YoY em todas as secoes
- [ ] Report revisado e formatado
- [ ] Distribuido para stakeholders
