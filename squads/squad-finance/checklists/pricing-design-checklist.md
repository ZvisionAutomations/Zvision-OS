# Pricing Design Checklist

## 1. Fundamento de Custo
- [ ] Custo por unidade de delivery calculado (cost-per-delivery)
- [ ] Custo loaded da equipe atualizado (ultimo 3 meses)
- [ ] Floor price definido (custo + margem minima 40%)
- [ ] Overhead alocado proporcionalmente
- [ ] Contingencia de 15-20% incluida no custo

## 2. Analise de Mercado
- [ ] Pelo menos 5 concorrentes mapeados com precos
- [ ] Precos normalizados para comparacao justa
- [ ] Posicionamento de preco definido (below/at/above market)
- [ ] Faixa aceitavel de preco identificada

## 3. Fundamento de Valor
- [ ] Valor entregue ao cliente quantificado em R$
- [ ] Hormozi Value Equation aplicada (Dream × Likelihood / Time × Effort)
- [ ] ROI projetado para o cliente calculado
- [ ] Narrativa de valor construida e convincente

## 4. Arquitetura de Tiers
- [ ] 3 tiers definidos (Good/Better/Best)
- [ ] Atributos diferenciadores claros entre tiers
- [ ] Better e claramente o melhor custo-beneficio
- [ ] Proporcoes de preco psicologicas (Good:Better = 1:2, Better:Best = 1:1.7)
- [ ] Conversao esperada: 20-30% Good, 50-60% Better, 10-20% Best
- [ ] Cada tier tem margem acima do minimo

## 5. Simulacao de Impacto
- [ ] 3 cenarios modelados (conservador, balanceado, premium)
- [ ] Sensibilidade de volume simulada (+/- 20%)
- [ ] Break-even volume calculado
- [ ] Impacto no P&L projetado para 12 meses
- [ ] Receita projetada por cenario

## 6. Politica Comercial
- [ ] Politica de descontos definida com limites
- [ ] Aprovacoes necessarias por nivel de desconto
- [ ] Floor de preco (NUNCA abaixo de 2× custo loaded)
- [ ] Condicoes de pagamento padrao definidas
- [ ] Early payment discount definido (se aplicavel)

## 7. Rate Card (se aplicavel)
- [ ] Rate card por role e senioridade calculado
- [ ] Markup minimo de 2.5× custo loaded
- [ ] Comparacao com mercado realizada
- [ ] Blended rate calculado para estimativas

## 8. Validacao Final
- [ ] Pricing validado pelo finance-orqx (Ledger)
- [ ] Margem global acima do target
- [ ] Alinhamento com estrategia de posicionamento
- [ ] Material de suporte para equipe comercial preparado
- [ ] FAQ de objecoes de preco mapeado
