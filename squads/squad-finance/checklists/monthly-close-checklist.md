# Monthly Close Checklist

## 1. Receita e Faturamento (Flow)
- [ ] Todas as faturas do mes emitidas
- [ ] Faturas reconciliadas com pagamentos recebidos
- [ ] Aging de contas a receber atualizado
- [ ] DSO calculado e comparado com target (<=45 dias)
- [ ] Collection rate do mes calculado (>=90%)
- [ ] MRR/ARR atualizado com new, expansion, contraction, churn
- [ ] Revenue recognition aplicado por contrato (competencia, nao caixa)
- [ ] Receita diferida e accrued reconciliadas

## 2. Custos e Despesas (Margin + Vault)
- [ ] Timesheets de toda a equipe coletados e validados
- [ ] Custo loaded atualizado (se houver mudanca de folha)
- [ ] Despesas diretas de projeto alocadas
- [ ] Freelancers e terceirizados contabilizados
- [ ] Ferramentas e SaaS consolidados
- [ ] Overhead alocado conforme metodo definido
- [ ] Despesas classificadas por cost center

## 3. P&L (Margin)
- [ ] Receita liquida calculada (bruta - impostos)
- [ ] COGS consolidado
- [ ] Gross margin calculado e comparado com target (>=50%)
- [ ] Despesas operacionais consolidadas
- [ ] EBITDA calculado e comparado com target (>=15%)
- [ ] Comparacao MoM realizada
- [ ] Comparacao YoY realizada (se historico disponivel)
- [ ] Comparacao vs budget realizada

## 4. Budget (Vault)
- [ ] Budget vs actual por rubrica
- [ ] Variancia calculada e classificada (verde/amarelo/vermelho)
- [ ] Root cause para variancias > 10%
- [ ] Pacing YTD atualizado
- [ ] EAC projetado

## 5. Pricing (Mint)
- [ ] Rate realization do mes calculada
- [ ] Average deal size atualizado
- [ ] Win rate atualizado (se dados disponiveis)
- [ ] Descontos concedidos no mes registrados

## 6. Cash Flow (Vault)
- [ ] Saldo bancario conferido
- [ ] Cash flow do mes calculado
- [ ] Runway atualizado (>=6 meses)
- [ ] Projecao de caixa para proximo mes

## 7. Dashboard Executivo (Ledger)
- [ ] Todos os KPIs coletados e consolidados
- [ ] Traffic light aplicado a cada KPI
- [ ] Executive summary 3x3 redigido
- [ ] Alertas triados e priorizados
- [ ] Top 3 recomendacoes definidas
- [ ] Dashboard formatado e distribuido

## 8. Compliance e Documentacao
- [ ] Todos os arquivos salvos com naming convention
- [ ] Dados sao consistentes entre reports
- [ ] Report revisado por pelo menos 1 pessoa
- [ ] Distribuido ate D+10
