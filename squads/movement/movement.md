# Movement Squad

7 agentes para construir movimentos que transcendem marcas e produtos.

## Inicio Rapido

```
@movement-chief   # Ativar o orquestrador
*diagnose         # Triagem do seu desafio de movimento
*build-movement   # Workflow completo de lancamento de movimento
```

## Agentes (7)

- **movement-chief** — Orquestrador
- **movement-architect** — Design de movimento
- **fenomenologo** — Analise de experiencia compartilhada
- **identitario** — Formacao de identidade
- **estrategista-de-ciclo** — Ciclos de crescimento
- **manifestador** — Criacao de manifestos
- **analista-de-impacto** — Medicao de impacto

## Componentes

- **7 agentes**, **7 tasks**, **1 workflow**, **1 checklist**
