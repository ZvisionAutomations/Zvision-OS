# Extraction Quality Checklist

> Valida completude e qualidade das extracoes cognitivas antes de prosseguir para sintese.

---

## Cobertura de Layers

- [ ] Layer 1 (Mental Models) extraida com min requisitos do tier
- [ ] Layer 2 (Heuristics) extraida com min requisitos do tier
- [ ] Layer 3 (Workflows) extraida com min requisitos do tier
- [ ] Layer 4 (Communication) extraida (vocabulario, tom, metaforas)
- [ ] Layer 5 (Meta-patterns) tentada (pode ser vazia em Tier 1)

## Tags de Confianca

- [ ] TODA extracao tem tag: [DIRETO], [INFERIDO] ou [HIPOTESE]
- [ ] [DIRETO] inclui citacao/fonte especifica
- [ ] [INFERIDO] referencia 3+ fontes de suporte
- [ ] [HIPOTESE] esta claramente marcada como precisando validacao
- [ ] Nenhuma extracao sem tag

## Contradicoes

- [ ] Contradicoes entre fontes identificadas e documentadas
- [ ] Cada contradicao tem contexto (quando, onde, para quem)
- [ ] Hipotese de resolucao proposta (evolucao? contexto? inconsistencia?)

## Honestidade

- [ ] Nenhuma extracao foi inventada/fabricada
- [ ] Gaps documentados explicitamente (o que nao foi encontrado)
- [ ] Nenhum [HIPOTESE] foi promovido a [INFERIDO] sem evidencia adicional

## Minimos por Tier

### Tier 1
- [ ] ≥5 mental models
- [ ] ≥8 heuristics
- [ ] ≥3 workflows
- [ ] ≥5 content patterns

### Tier 2 (adicional)
- [ ] ≥10 mental models
- [ ] ≥15 heuristics
- [ ] ≥5 workflows
- [ ] ≥8 content patterns
- [ ] ≥3 contradicoes documentadas

### Tier 3 (adicional)
- [ ] ≥15 mental models
- [ ] ≥25 heuristics
- [ ] ≥8 workflows
- [ ] ≥12 content patterns
- [ ] ≥10 decision rules
- [ ] Vocabulario/tom extraido
- [ ] ≥5 meta-patterns

## Extraction Report

- [ ] Relatorio gerado com estatisticas por layer
- [ ] Distribuicao de confidence tags documentada
- [ ] Gaps e recomendacoes listados
