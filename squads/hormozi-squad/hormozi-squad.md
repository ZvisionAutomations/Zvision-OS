# Hormozi Squad

16 agentes especializados implementando os frameworks de escalamento de negocios do Alex Hormozi.

## Inicio Rapido

```
@hormozi-chief       # Ativar o orquestrador
*diagnose            # Triagem do seu desafio de negocio
*business-turnaround # Workflow completo de turnaround de negocio
*offer-pipeline      # Criacao de oferta ponta a ponta
```

- **hormozi-chief** — Orquestrador
- **hormozi-offers** — Criacao de Grand Slam Offer
- **hormozi-leads** — Framework $100M Leads
- **hormozi-pricing** — Precificacao baseada em valor
- **hormozi-closer** — Framework CLOSER & vendas
- **hormozi-ads** — Estrategia de anuncios pagos
- **hormozi-content** — Maquina de conteudo
- **hormozi-hooks** — Criacao de hooks
- **hormozi-launch** — Estrategia de lancamento
- **hormozi-retention** — Reducao de churn & LTV
- **hormozi-scale** — Escalando de $1M para $100M+
- **hormozi-models** — Design de modelo de negocio
- **hormozi-audit** — Avaliacao de negocio
- **hormozi-copy** — Copy estilo Hormozi
- **hormozi-workshop** — Design de workshops
- **hormozi-advisor** — Aconselhamento estrategico
