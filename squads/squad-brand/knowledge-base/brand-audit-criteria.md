# Brand Audit Criteria — Knowledge Base

> Critérios, métricas e frameworks para auditoria e diagnóstico de marca.

---

## 1. Framework de Auditoria — 5 Dimensões

### Dimensão 1: Consistência Visual (Peso: 30%)
| # | Critério | Descrição | Scoring |
|---|----------|-----------|---------|
| 1.1 | Logo usage | Logo aplicado corretamente em todos os touchpoints | 0-5 |
| 1.2 | Color consistency | Cores dentro da paleta definida, ratios respeitados | 0-5 |
| 1.3 | Typography | Fontes corretas, hierarquia respeitada | 0-5 |
| 1.4 | Photography | Estilo fotográfico alinhado ao brand book | 0-5 |
| 1.5 | Graphic elements | Patterns, ícones, ilustrações consistentes | 0-5 |
| 1.6 | Layout & grid | Grid system e espaçamentos respeitados | 0-5 |

### Dimensão 2: Alinhamento Estratégico (Peso: 25%)
| # | Critério | Descrição | Scoring |
|---|----------|-----------|---------|
| 2.1 | Positioning | Posicionamento refletido nas comunicações | 0-5 |
| 2.2 | Tone of voice | Tom consistente e alinhado ao manifesto | 0-5 |
| 2.3 | Message alignment | Mensagens-chave presentes e claras | 0-5 |
| 2.4 | Audience fit | Comunicação adequada ao público-alvo | 0-5 |
| 2.5 | Differentiation | Diferenciação clara vs concorrentes | 0-5 |
| 2.6 | Values expression | Valores da marca visíveis nas ações | 0-5 |

### Dimensão 3: Qualidade Técnica (Peso: 20%)
| # | Critério | Descrição | Scoring |
|---|----------|-----------|---------|
| 3.1 | Resolution | Assets em resolução adequada por uso | 0-5 |
| 3.2 | File formats | Formatos corretos (CMYK print, RGB digital) | 0-5 |
| 3.3 | Accessibility | WCAG AA compliance (contraste, alt text) | 0-5 |
| 3.4 | Responsiveness | Assets adaptam-se a diferentes telas | 0-5 |
| 3.5 | Performance | Otimização de peso/carregamento | 0-5 |
| 3.6 | Code quality | Tokens, componentes, CSS bem estruturados | 0-5 |

### Dimensão 4: Cross-Channel (Peso: 15%)
| # | Critério | Descrição | Scoring |
|---|----------|-----------|---------|
| 4.1 | Digital consistency | Site, app, email = mesma marca | 0-5 |
| 4.2 | Social consistency | Todas as redes = mesma marca | 0-5 |
| 4.3 | Print consistency | Materiais impressos = mesma marca | 0-5 |
| 4.4 | Environmental | Sinalização, ambientes = mesma marca | 0-5 |
| 4.5 | Audio-visual | Vídeo, áudio = mesma marca | 0-5 |

### Dimensão 5: Documentação (Peso: 10%)
| # | Critério | Descrição | Scoring |
|---|----------|-----------|---------|
| 5.1 | Brandbook | Existe, está completo e atualizado | 0-5 |
| 5.2 | Quick reference | Card de referência rápida disponível | 0-5 |
| 5.3 | Asset library | Biblioteca organizada e acessível | 0-5 |
| 5.4 | Governance | Processo de aprovação definido e seguido | 0-5 |
| 5.5 | Training | Equipe conhece e segue as guidelines | 0-5 |

## 2. Sistema de Scoring

### Escala Individual (0-5)
| Score | Label | Descrição |
|-------|-------|-----------|
| 5 | Excelente | Excede expectativas, referência de mercado |
| 4 | Bom | Atende completamente às guidelines |
| 3 | Adequado | Atende parcialmente, melhorias menores |
| 2 | Insuficiente | Problemas significativos, requer ação |
| 1 | Crítico | Falhas graves, ação urgente |
| 0 | Ausente | Não existe ou completamente inadequado |

### Cálculo do Brand Consistency Score (BCS)

```
BCS = (Visual × 0.30) + (Strategic × 0.25) + (Technical × 0.20) + (CrossChannel × 0.15) + (Documentation × 0.10)
```

Onde cada dimensão é a média dos seus critérios (0-5), resultando em BCS de 0-5.

### Conversão para Porcentagem
```
BCS% = (BCS / 5) × 100
```

### Thresholds de Decisão
| BCS% | Status | Ação |
|------|--------|------|
| 85-100% | Healthy | Manutenção preventiva |
| 70-84% | Attention | Correções pontuais planejadas |
| 50-69% | Intervention | Plano de ação corretivo urgente |
| < 50% | Critical | Redesign/rebranding recomendado |

## 3. Diagnóstico de Misalignment

### Sintomas Comuns
| Sintoma | Possível Causa Raiz | Impacto |
|---------|---------------------|---------|
| Logo inconsistente entre canais | Falta de asset library centralizada | Confusão de identidade |
| Cores diferentes em materiais | Falta de paleta documentada / perfil de cor | Diluição visual |
| Tom de voz variável | Múltiplos redatores sem guia | Personalidade inconsistente |
| Materiais desatualizados | Governança fraca, sem versionamento | Marca fragmentada |
| Desconexão digital vs print | Guidelines não cobrem todos os meios | Experiência desalinhada |
| Feedback negativo de clientes | Posicionamento não refletido na prática | Perda de confiança |

### Framework de Análise: Causa → Impacto → Correção
```
1. IDENTIFICAR sintoma (o que está errado)
2. DIAGNOSTICAR causa raiz (por que está errado)
3. AVALIAR impacto (quão grave é)
4. PRESCREVER correção (o que fazer)
5. PREVENIR recorrência (como evitar no futuro)
```

## 4. Benchmarking

### Métricas Comparativas
| Métrica | Como Medir | Benchmark Bom |
|---------|-----------|---------------|
| Brand recall | Survey: "Que marca você lembra de [categoria]?" | Top 3 menções |
| Visual recognition | Teste: reconhece logo sem nome? | >80% do público |
| Message clarity | Survey: "O que essa marca faz?" | >70% resposta correta |
| Differentiation | "O que diferencia de [concorrente]?" | >50% articulam |
| Consistency perception | "A marca parece a mesma em todos os canais?" | >85% concordam |
| NPS (brand-related) | Net Promoter Score | >50 |

### Análise Competitiva Visual
| Critério | Avaliar | Escala |
|----------|---------|--------|
| Distinção cromática | Paleta vs top 5 concorrentes | Única / Similar / Idêntica |
| Distinção tipográfica | Fontes vs top 5 concorrentes | Única / Similar / Idêntica |
| Distinção de estilo | Visual language vs mercado | Líder / Par / Seguidor |
| Share of voice visual | Presença vs concorrentes | Alta / Média / Baixa |

## 5. Auditoria por Touchpoint

|------------|--------------|
| Website | Logo, cores, tipografia, imagery, UX, performance, SEO |

| Social profiles | Avatar, cover, bio, tom de voz, visual feed |

| Brochures/folders | Layout, imagery, copy, acabamento |

## 6. Relatório de Auditoria — Estrutura

### Seções do Relatório
1. **Executive Summary** — Score geral, top issues, recomendações prioritárias
2. **Methodology** — O que foi avaliado, como, critérios usados
3. **Scores por Dimensão** — Gráfico radar + detalhamento
4. **Top Findings** — 5-10 achados mais relevantes
5. **Touchpoint Analysis** — Avaliação por canal/touchpoint
6. **Competitive Context** — Comparação com concorrentes
7. **Recommendations** — Priorizadas por impacto × esforço (matriz 2x2)
8. **Action Plan** — Timeline, responsáveis, recursos necessários

### Priorização (Matriz Impacto × Esforço)
| Quadrante | Impacto | Esforço | Ação |
|-----------|---------|---------|------|
| Quick Wins | Alto | Baixo | Fazer primeiro |
| Strategic | Alto | Alto | Planejar com cuidado |
| Fill-ins | Baixo | Baixo | Fazer quando possível |
| Thankless | Baixo | Alto | Evitar ou adiar |

## 7. Frequência de Auditoria

| Tipo | Frequência | Escopo |
|------|-----------|--------|
| Full audit | Anual | Todas as 5 dimensões, todos os touchpoints |
| Quick check | Trimestral | Top issues + novos materiais |
| Post-launch | Após cada campanha | Materiais da campanha vs guidelines |
| Trigger-based | Quando necessário | Após rebranding, crise, fusão |
| Continuous | Automatizado | Monitoramento digital (cores, fontes, logos em uso) |
