## 2. Diretrizes de Mockup por Categoria

### Template de Prompt para Geração
```
[CATEGORY]: [SPECIFIC PRODUCT]
Brand: [BRAND NAME]
Colors: [PRIMARY HEX], [SECONDARY HEX], [ACCENT HEX]
Logo: [DESCRIPTION OR "overlay in post"]
Style: [PRODUCT CATEGORY STYLE]
View: [ANGLE/PERSPECTIVE]
Background: [SURFACE/CONTEXT]
Quality: [RESOLUTION, ENGINE]
```

### Apparel — Regras
- Logo deve ser legível a 2m de distância
- Mockup em modelo: diversidade de tons de pele
- Flat lay: casting de sombra natural
- Tag interna: marca + material + tamanho
- Embalagem: papel tissue + adesivo de marca

### Stationery — Regras
- Sempre mostrar frente E verso
- Business card: mínimo 3 composições (stack, spread, holder)
- Letterhead: mostrar com texto simulado preenchido
- Envelope: mostrar lacrado + endereçado
- Print-ready: bleed 3mm, safe zone 5mm, CMYK

### Packaging — Regras
- Sempre incluir dieline técnico
- Mockup 3D + foto real se possível
- Informações legais visíveis mas discretas
- Código de barras/QR na posição correta
- Materiais sustentáveis preferidos (quando possível)

### Digital — Regras
- Device mockup atualizado (não usar iPhone 6 em 2026)
- UI legível no mockup (não apenas decorativa)
- Mostrar estado real da interface (com dados)
- Mobile-first: mostrar mobile antes de desktop
- Dark mode: incluir versão se aplicável

## 5. Checklist de Mockup

- [ ] Logo aplicado corretamente (área de proteção, tamanho mínimo)
- [ ] Cores corretas (verificar HEX/CMYK)
- [ ] Tipografia correta (fonte certa, hierarquia)
- [ ] Proporções realistas (tamanho do produto correto)
- [ ] Iluminação consistente em toda a série
- [ ] Sombras naturais e coerentes
- [ ] Background adequado ao uso
- [ ] Resolução adequada ao destino
- [ ] Formato correto (RGB digital, CMYK print)
- [ ] Variações necessárias incluídas (frente/verso, light/dark)
