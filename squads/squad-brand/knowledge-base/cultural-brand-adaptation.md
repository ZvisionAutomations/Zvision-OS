## 1. Frameworks de Dimensões Culturais

### Hofstede — 6 Dimensões

| Dimensão | Low Score | High Score | Implicação para Branding |
|----------|-----------|------------|--------------------------|
| **Power Distance** | Dinamarca 18, Israel 13 | Malásia 100, Eslováquia 100 | Tom: igualitário ("somos parceiros") vs hierárquico ("confie nos especialistas") |
| **Individualism** | Guatemala 6, Equador 8 | EUA 91, Austrália 90 | Messaging: "você" + autonomia vs "nós" + pertencimento |
| **Masculinity** | Suécia 5, Noruega 8 | Japão 95, Hungria 88 | Valores: competição + achievement vs cooperação + quality of life |
| **Uncertainty Avoidance** | Singapura 8, Jamaica 13 | Grécia 100, Portugal 99 | Confiança: dados, garantias, certificações vs abertura ao novo |
| **Long-term Orientation** | Gana 4, Colômbia 13 | Coreia do Sul 100, Taiwan 93 | Narrativa: tradição + investimento futuro vs resultado imediato |
| **Indulgence** | Paquistão 0, Egito 4 | Venezuela 100, México 97 | Visual: expressivo + emocional vs contido + funcional |

### Erin Meyer — 8 Escalas Culturais

| Escala | Um Extremo | Outro Extremo | Implicação |
|--------|-----------|--------------|-----------|
| Comunicar | Low-context (EUA, Holanda) | High-context (Japão, Coreia) | Copy: explícito + direto vs sutil + implícito |
| Avaliar | Feedback direto (Holanda, Rússia) | Feedback indireto (Japão, Tailândia) | Reviews: transparentes vs diplomáticas |
| Persuadir | Principles-first (França, Itália) | Applications-first (EUA, Canadá) | Conteúdo: "por que" primeiro vs "como" primeiro |
| Liderar | Egalitarian (Dinamarca, Suécia) | Hierarchical (Japão, Nigéria) | Marca como: amigo vs autoridade |
| Decidir | Consensual (Japão, Suécia) | Top-down (China, Rússia) | CTA: "junte-se a nós" vs "escolha agora" |
| Confiar | Task-based (EUA, Holanda) | Relationship-based (China, Brasil) | Jornada: direto ao produto vs construir relação |
| Discordar | Confrontational (França, Israel) | Avoid confrontation (Japão, Indonésia) | Competição: comparativo direto vs diferenciação sutil |
| Agendar | Linear-time (Alemanha, Suíça) | Flexible-time (Índia, Nigéria) | Urgência: deadlines + precisão vs flexibilidade |

## 3. Modelo Glocal — Implementação

### Níveis de Adaptação

```
Nível 1: TRADUÇÃO
  └─ Apenas texto traduzido. Visual idêntico.
     Exemplo: Manual de produto Apple

Nível 2: LOCALIZAÇÃO
  └─ Texto + imagens + formatos adaptados.
     Exemplo: Website Airbnb (fotos locais, moeda, idioma)

Nível 3: ADAPTAÇÃO
  └─ Conceito mantido, execução local.
     Exemplo: Campanhas McDonald's (produto local + visual global)

Nível 4: CRIAÇÃO LOCAL
  └─ Conteúdo original dentro dos guidelines.
     Exemplo: Netflix originals por país

Nível 5: MARCA SEPARADA
  └─ Brand diferente por mercado.
     Exemplo: Axe/Lynx, TK Maxx/TJ Maxx
```

### Checklist de Localização por Touchpoint

| Touchpoint | Nível Mínimo | Verificações |
|-----------|-------------|-------------|
| Website | 2 | Idioma, moeda, imagens, SEO local, LGPD/GDPR |
| App | 2 | RTL, formatos locais, payment methods |
| Social media | 3-4 | Plataformas locais (WeChat, LINE), trends, timing |
| Embalagem | 2-3 | Regulatório, idioma, unidades de medida |
| Publicidade | 3 | Modelos locais, contexto cultural, regulamentação |
| Atendimento | 3 | Idioma, protocolos culturais, fuso horário |

## 4. Casos de Referência — Erros e Acertos

### Erros Famosos
| Marca | Mercado | Erro | Lição |
|-------|---------|------|-------|
| Pepsi | China | "Come alive" → "traz ancestrais do túmulo" | Validar tradução com nativos |
| KFC | China | "Finger lickin' good" → "coma seus dedos" | Adaptar, não traduzir literalmente |
| Gerber | África | Foto de bebê na embalagem → medo | Entender convenções visuais locais |
| Chevrolet | LatAm | Nova = "não vai" em espanhol | Testar naming em todos os idiomas |
| HSBC | Global | "Assume nothing" → "do nothing" | Custo: $10M para rebrand "The world's local bank" |

### Acertos de Referência
| Marca | Mercado | Estratégia | Resultado |
|-------|---------|-----------|----------|
| McDonald's | Índia | McAloo Tikki, menu sem beef | Maior crescimento na região |
| Coca-Cola | China | 可口可乐 = "deliciosa felicidade" | Nome transcriado, não traduzido |
| Kit Kat | Japão | 300+ sabores regionais + "kitto katsu" (vai vencer) | Presente cultural de boa sorte |
| IKEA | Global | Showrooms adaptados ao tamanho local de moradia | Relevância contextual |
| Spotify | MENA | Playlists para Ramadan, artistas locais | Respeito cultural + relevância |

## 5. Adaptação Tipográfica

### Pares Tipográficos por Script

| Script | Fonte Recomendada | Pairing com Latin | Notas |
|--------|------------------|-------------------|-------|
| **CJK** (Chinês, Japonês, Coreano) | Noto Sans CJK, Source Han Sans | Combinar x-height | Sem serif para UI digital |
| **Árabe** | Noto Sans Arabic, IBM Plex Arabic | Combinar peso visual | RTL, ligatures |
| **Devanagari** (Hindi) | Noto Sans Devanagari, Poppins | Combinar x-height | Shirorekha (headline) |
| **Cirílico** | Inter, Golos Text | Mesma família se disponível | Verificar caracteres especiais |
| **Thai** | Noto Sans Thai, Sarabun | Combinar ascender height | Sem bold para body text |
| **Hebraico** | Noto Sans Hebrew, Heebo | RTL layout | Diacríticos opcionais |

### Considerações de Layout

| Idioma | Expansão vs Inglês | Consideração |
|--------|-------------------|-------------|
| Alemão | +30% | Compostos longos (Rechtsschutzversicherungsgesellschaften) |
| Russo | +20% | Palavras mais longas em geral |
| Chinês | -30% | Mais denso, menos espaço necessário |
| Árabe | +25% | RTL completo, pontuação invertida |
| Japonês | -20% a +10% | Misto vertical/horizontal |
| Português (BR) | +15% | Acentuação, preposições |
| Francês | +20% | Artigos, conjugações |

## 6. Compliance Regulatório por Região

| Região | Regulamentação | Impacto em Branding |
|--------|---------------|-------------------|
| **EU** | GDPR, EU Accessibility Act | Consent, cookie banners, WCAG AA |
| **Brasil** | LGPD, CONAR | Privacidade, publicidade responsável |
| **EUA** | FTC, ADA, CCPA (CA) | Disclaimers, accessibility, privacy |
| **China** | CSL, PIPL, CAL | Censura, dados locais, anti-extravagância |
| **Oriente Médio** | Sharia compliance | Modéstia, halal, conteúdo religioso |
| **Índia** | ASCI, IT Act | Publicidade comparativa, dados |
| **Japão** | JARO, APPI | Não-comparativa, privacy rigorosa |
| **Austrália** | ACCC, Privacy Act | Anti-misleading, consent |
