# Archetype-Brand Mapping — Knowledge Base

> Mapeamento completo dos 12 arquétipos de marca para estratégia, visual e comunicação.

---

## 1. Os 12 Arquétipos de Marca

Baseado no framework de Carl Jung, adaptado para branding por Margaret Mark e Carol S. Pearson.

### Quadrantes
| Motivação | Arquétipos |
|-----------|-----------|
| **Estabilidade & Controle** | Cuidador, Governante, Criador |
| **Pertencimento & Prazer** | Cara Comum, Bobo, Amante |
| **Risco & Maestria** | Herói, Fora-da-lei, Mago |
| **Independência & Realização** | Inocente, Explorador, Sábio |

---

## 2. Perfil Completo por Arquétipo

### 2.1 Inocente (The Innocent)
| Aspecto | Definição |
|---------|-----------|
| **Desejo core** | Ser feliz, viver em harmonia |
| **Medo** | Fazer algo errado, ser punido |
| **Estratégia** | Fazer as coisas certas |
| **Mensagem** | "A vida pode ser simples e boa" |
| **Tom de voz** | Otimista, simples, sincero, puro |
| **Cores** | Branco, azul claro, amarelo suave, pastéis |
| **Tipografia** | Sans-serif clean, rounded, friendly |
| **Fotografia** | Luz natural, sorriso genuíno, natureza |
| **Marcas exemplo** | Dove, Coca-Cola (classic), Nintendo |
| **Setores** | Alimentos orgânicos, cuidados pessoais, infantil |

### 2.2 Explorador (The Explorer)
| Aspecto | Definição |
|---------|-----------|
| **Desejo core** | Liberdade para descobrir e se encontrar |
| **Medo** | Ficar preso, conformismo |
| **Estratégia** | Aventurar-se, experimentar coisas novas |
| **Mensagem** | "Não se limite, explore" |
| **Tom de voz** | Aventureiro, autêntico, independente |
| **Cores** | Verde escuro, terracota, azul profundo, terra |
| **Tipografia** | Sans-serif robusta, display com personalidade |
| **Fotografia** | Paisagens grandiosas, outdoor, aventura |
| **Marcas exemplo** | Patagonia, Jeep, The North Face, Airbnb |
| **Setores** | Outdoor, viagens, automóveis, lifestyle |

### 2.3 Sábio (The Sage)
| Aspecto | Definição |
|---------|-----------|
| **Desejo core** | Compreender o mundo pela verdade e conhecimento |
| **Medo** | Ignorância, ser enganado |
| **Estratégia** | Buscar informação e conhecimento |
| **Mensagem** | "A verdade vai te libertar" |
| **Tom de voz** | Intelectual, informativo, confiável, analítico |
| **Cores** | Azul marinho, cinza, branco, dourado sóbrio |
| **Tipografia** | Serif clássica, pesos variados, hierarquia clara |
| **Fotografia** | Minimalista, studio, dados visuais |
| **Marcas exemplo** | Google, TED, The Economist, Harvard |
| **Setores** | Educação, consultoria, tech, mídia |

### 2.4 Herói (The Hero)
| Aspecto | Definição |
|---------|-----------|
| **Desejo core** | Provar seu valor através de ação corajosa |
| **Medo** | Fraqueza, vulnerabilidade |
| **Estratégia** | Ser forte, competente, poderoso |
| **Mensagem** | "Onde há vontade, há um caminho" |
| **Tom de voz** | Bold, inspirador, determinado, direto |
| **Cores** | Vermelho, preto, azul escuro, contraste alto |
| **Tipografia** | Display bold, impactante, all-caps |
| **Fotografia** | Dinâmica, ação, contraste, empoderamento |
| **Marcas exemplo** | Nike, FedEx, BMW, Under Armour |
| **Setores** | Esporte, automotivo, serviços premium |

### 2.5 Fora-da-lei (The Outlaw)
| Aspecto | Definição |
|---------|-----------|
| **Desejo core** | Revolução, quebrar regras |
| **Medo** | Impotência, irrelevância |
| **Estratégia** | Disrupção, destruição criativa |
| **Mensagem** | "Regras existem para serem quebradas" |
| **Tom de voz** | Provocativo, rebelde, irreverente |
| **Cores** | Preto, vermelho escuro, roxo, metalizado |
| **Tipografia** | Display condensada, grunge, industrial |
| **Fotografia** | Alto contraste, grão, urban, raw |
| **Marcas exemplo** | Harley-Davidson, Diesel, Virgin |
| **Setores** | Moda alternativa, entretenimento, tech disruptiva |

### 2.6 Mago (The Magician)
| Aspecto | Definição |
|---------|-----------|
| **Desejo core** | Entender leis fundamentais do universo |
| **Medo** | Consequências negativas não intencionais |
| **Estratégia** | Desenvolver visão e vivê-la |
| **Mensagem** | "Tornar sonhos em realidade" |
| **Tom de voz** | Visionário, transformador, misterioso |
| **Cores** | Roxo, dourado, azul profundo, preto com brilho |
| **Tipografia** | Display elegante, serifas refinadas |
| **Fotografia** | Surreal, efeitos especiais, transformação |
| **Marcas exemplo** | Apple, Disney, Tesla |
| **Setores** | Tech inovadora, entretenimento, luxo |

### 2.7 Cara Comum (The Everyman)
| Aspecto | Definição |
|---------|-----------|

| **Fotografia** | Real, candid, diverso, cotidiano |

| **Setores** | Varejo, alimentos, serviços do dia-a-dia |

### 2.8 Amante (The Lover)
| Aspecto | Definição |
|---------|-----------|
| **Desejo core** | Intimidade e experiência sensorial |
| **Medo** | Ser indesejado, não atrativo |
| **Estratégia** | Tornar-se atraente física e emocionalmente |

| **Tipografia** | Serif elegante, script refinada, thin weights |
| **Fotografia** | Close-up, luz suave, texturas, luxo |
| **Marcas exemplo** | Chanel, Victoria's Secret, Godiva |
| **Setores** | Moda, cosmetics, joias, gastronomia fina |

### 2.9 Bobo (The Jester)
| Aspecto | Definição |
|---------|-----------|
| **Desejo core** | Viver o momento com alegria |
| **Medo** | Tédio, ser chato |
| **Estratégia** | Ser divertido, brincar |

| **Tipografia** | Display playful, rounded, bouncy |
| **Fotografia** | Colorida, pop, dinâmica, expressiva |

### 2.10 Cuidador (The Caregiver)

| **Desejo core** | Proteger e cuidar dos outros |
| **Medo** | Egoísmo, ingratidão |
| **Estratégia** | Fazer pelos outros |
| **Mensagem** | "Ame seu próximo como a si mesmo" |

| **Cores** | Azul suave, verde, branco, tons quentes |

| **Fotografia** | Humana, conexão, sorrisos, cuidado |

| Aspecto | Definição |
|---------|-----------|
| **Desejo core** | Criar algo de valor duradouro |
| **Medo** | Mediocridade, visão sem execução |
| **Estratégia** | Desenvolver habilidade artística |
| **Mensagem** | "Se pode ser imaginado, pode ser criado" |

| **Setores** | Arte, design, ferramentas criativas, artesanal |

### 2.12 Governante (The Ruler)
| Aspecto | Definição |
|---------|-----------|
| **Desejo core** | Controle, poder, sucesso |

| **Tipografia** | Serif clássica, all-caps, espaçamento amplo |
| **Fotografia** | Grandiosa, simétrica, poder, luxo |
| **Marcas exemplo** | Rolex, Mercedes-Benz, American Express |
| **Setores** | Luxo, financeiro, hotelaria premium, automotive |

## 3. Combinação de Arquétipos

### Regras
1. **Primário + Secundário** — Um arquétipo dominante + um influência secundária
2. **Máximo 2** — Mais de 2 dilui a personalidade
3. **Compatibilidade** — Escolher arquétipos do mesmo quadrante ou adjacentes
4. **Proporção** — 70% primário / 30% secundário

### Combinações Comuns
| Primário | Secundário | Resultado | Exemplo |
|----------|-----------|-----------|---------|
| Herói | Sábio | Competência + expertise | Nike Academy |
| Mago | Criador | Inovação + artesanato | Apple |
| Explorador | Fora-da-lei | Liberdade + rebeldia | Harley-Davidson |
| Amante | Governante | Desejo + exclusividade | Chanel |
| Cuidador | Sábio | Proteção + conhecimento | Mayo Clinic |
| Bobo | Inocente | Diversão + pureza | Ben & Jerry's |

### Combinações a Evitar
| Combinação | Por que Evitar |
|-----------|---------------|
| Herói + Bobo | Mensagens contraditórias (sério × divertido) |
| Governante + Cara Comum | Exclusividade × acessibilidade |
| Inocente + Fora-da-lei | Pureza × rebeldia |
| Cuidador + Fora-da-lei | Proteção × destruição |

## 4. Mapeamento Arquétipo → Design System

### Quick Reference
| Arquétipo | Border Radius | Shadow | Animation | Spacing |
|-----------|--------------|--------|-----------|---------|
| Inocente | Rounded (16-24px) | Soft | Gentle, bounce | Generous |
| Explorador | Medium (8-12px) | Natural | Dynamic, parallax | Balanced |
| Sábio | Sharp (0-4px) | Minimal | Subtle, precise | Structured |
| Herói | Medium-sharp (4-8px) | Strong | Bold, impactful | Tight |
| Fora-da-lei | Mixed/broken | Harsh | Glitch, unexpected | Tight |
| Mago | Varied | Glow | Magical, morphing | Airy |
| Cara Comum | Medium (8px) | Standard | Simple, functional | Standard |
| Amante | Soft (12-20px) | Warm | Smooth, sensual | Generous |
| Bobo | Full round/organic | Playful | Bouncy, exaggerated | Loose |
| Cuidador | Rounded (12-16px) | Soft | Gentle, nurturing | Comfortable |
| Criador | Eclectic | Artistic | Crafted, detailed | Dynamic |
| Governante | Sharp (0-2px) | Elegant | Stately, controlled | Precise |
