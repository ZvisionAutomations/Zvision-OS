# Brand Valuation Frameworks — Knowledge Base

> Metodologias de valuação de marca para cálculo de equity e ROI de branding.

---

## 1. Por Que Valuar uma Marca

A valuação de marca quantifica o valor financeiro gerado pela percepção da marca. No contexto do squad-brand, essa valuação é integrada a cada entrega para demonstrar:
- **Valor antes vs depois** de cada fase de branding
- **ROI do investimento** em branding vs valor gerado
- **Crescimento de equity** ao longo das entregas
- **Benchmark** contra marcas de referência no setor

---

## 2. Metodologias de Valuação

### 2.1 Cost-Based (Custo de Criação)
**O que mede:** Quanto custaria recriar a marca do zero.

**Componentes:**
| Item | Descrição | Range de Mercado |
|------|-----------|-----------------|
| Estratégia de marca | Discovery, posicionamento, manifesto | $5K-50K |
| Identidade visual | Logo, paleta, tipografia, guidelines | $5K-100K |
| Design system | Tokens, componentes, dark mode | $10K-80K |
| Assets criativos | Mockups, templates, colaterais | $5K-50K |
| Motion + Audio | Animações, sonic identity | $3K-30K |
| Brandbook + Documentação | Compilação, governança | $3K-20K |

**Fórmula:**
```
Brand Value (Cost) = Σ(custo de cada componente) × Fator de Qualidade (1.0-2.0)
```

**Fator de Qualidade:**
- 1.0: Execução básica
- 1.3: Execução profissional
- 1.5: Execução premium
- 1.8-2.0: Execução world-class (studio internacional)

### 2.2 Market-Based (Comparação de Mercado)
**O que mede:** Valor baseado em transações de marcas similares.

**Processo:**
1. Identificar 3-5 marcas similares no setor que foram vendidas/licenciadas
2. Calcular múltiplo de receita aplicado
3. Ajustar por tamanho, presença e maturidade

**Múltiplos de Referência por Setor:**
| Setor | Múltiplo Brand/Revenue | Exemplo |
|-------|----------------------|---------|
| Luxo/Moda | 2.0-5.0x | LVMH, Gucci |
| Tech/SaaS | 1.5-4.0x | Salesforce, HubSpot |
| Consumer Goods | 1.0-3.0x | P&G, Unilever |
| Food & Beverage | 1.0-2.5x | Coca-Cola, Starbucks |
| Services (B2B) | 0.5-2.0x | McKinsey, Accenture |
| Healthcare | 1.0-3.0x | Johnson & Johnson |
| Startup (Pre-revenue) | N/A | Usar Cost-Based |

### 2.3 Income-Based (Contribuição aos Lucros)
**O que mede:** Quanto a marca contribui para a receita/lucro.

**Método de Royalty Relief:**
```
Brand Value = Revenue × Royalty Rate × Tax Factor × Brand Strength Score × Discount Factor
```

**Royalty Rates por Setor:**
| Setor | Royalty Rate Típico |
|-------|-------------------|
| Moda/Luxo | 8-15% |
| Tech | 5-10% |
| Food & Beverage | 3-8% |
| Services | 3-7% |
| Healthcare | 5-10% |

**Brand Strength Score (BSS):**
Calculado em 10 dimensões (escala 0-100):

| Dimensão | Peso | Descrição |
|----------|------|-----------|
| Investimento em marca | 10% | Quanto investe em branding |
| Consistência | 15% | Quão consistente é cross-channel |
| Clareza interna | 10% | Equipe entende e vive a marca |
| Relevância | 15% | Relevante para público-alvo |
| Autenticidade | 10% | Cumpre o que promete |
| Diferenciação | 15% | Distinta dos concorrentes |
| Presença | 10% | Alcance e share of mind |
| Engagement | 10% | Conexão emocional |
| Proteção legal | 5% | Registro e proteção da marca |

### 2.4 Keller CBBE (Customer-Based Brand Equity)
**O que mede:** Percepção e lealdade do consumidor.

**Pirâmide de Brand Equity:**
```
Nível 4: RESSONÂNCIA (lealdade, comunidade, engagement)
    ↑
Nível 3: JULGAMENTO + SENTIMENTOS (qualidade, credibilidade, superioridade + warmth, fun, excitement)
    ↑
Nível 2: PERFORMANCE + IMAGÉTICA (características, confiabilidade, estilo + perfil de uso, personalidade)
    ↑
Nível 1: SALIÊNCIA (reconhecimento, recall, top-of-mind)
```

**Scoring por Nível:**
| Nível | Métrica | Como Medir | Score 0-25 |
|-------|---------|-----------|-----------|
| Saliência | Brand recall | Survey: "Cite marcas de [categoria]" | 0-25 |
| Significado | Associações | Survey: "O que [marca] significa para você?" | 0-25 |
| Resposta | Preferência | Survey: "Escolheria [marca] sobre alternativas?" | 0-25 |
| Ressonância | Lealdade | NPS + repeat purchase + advocacy | 0-25 |
| **Total** | **CBBE Score** | **Soma dos 4 níveis** | **0-100** |

## 3. Brand Value Tracker — Modelo de Cálculo

### Fase 0: Baseline (Antes do Projeto)
```
Baseline Score = Assessment dos assets existentes
├── Visual Identity Score: 0-100
├── Strategic Clarity Score: 0-100
├── Technical Quality Score: 0-100
├── Cross-Channel Score: 0-100
└── Documentation Score: 0-100

Baseline Brand Value = Cost-Based (assets existentes) + CBBE Score × Fator de Receita
```

### Fase 1: Post-Strategy
```
Added Value:
├── Positioning clarity: +{score_delta}
├── Brand architecture defined: +{value}
├── Manifesto created: +{value}
├── Tone of voice codified: +{value}
└── Personas mapped: +{value}

Phase 1 Brand Value = Baseline + Strategy Value Add
```

### Fase 2: Post-Visual
```
Added Value:
├── Logo system created: +{value}
├── Color palette defined: +{value}
├── Typography selected: +{value}
├── Visual language codified: +{value}

Phase 2 Brand Value = Phase 1 Value + Visual Value Add
```

## 4. Cálculo de ROI

### Fórmula de ROI
```
Brand ROI = ((Final Brand Value - Baseline Brand Value) / Investment Cost) × 100
```

### Exemplo
```
Baseline Brand Value: $15,000 (marca básica, logo amador, sem guidelines)
Investment (serviço de branding): $25,000
Final Brand Value: $120,000 (marca completa, world-class)

Value Added: $120,000 - $15,000 = $105,000
ROI: ($105,000 / $25,000) × 100 = 420% ROI
```

### Benchmarks de ROI por Tipo de Projeto
| Tipo | ROI Típico | Timeframe |
|------|-----------|-----------|
| Brand refresh | 200-400% | 6-12 meses |
| Full rebranding | 300-600% | 12-24 meses |
| Brand creation (startup) | 500-1000%+ | 12-36 meses |
| Design system | 150-300% | 6-18 meses |

## 6. Disclaimer e Rigor

### Premissas
- Valuação baseada em metodologias reconhecidas (Interbrand, Brand Finance, Keller CBBE)
- Valores são estimativas baseadas em benchmarks de mercado
- Comparação com transações reais do setor quando disponível
- Scores de brand strength baseados em avaliação qualitativa + quantitativa

### Fontes de Dados
- Relatórios Interbrand Best Global Brands
- Brand Finance Global 500
- Pesquisas setoriais de royalty rates
- Benchmarks de mercado para serviços de branding
- Dados de performance da marca (quando disponíveis: NPS, recall, traffic)

### Limitações
- Valuação não constitui avaliação financeira formal
- Valores de mercado podem variar significativamente
- Brand equity é influenciado por fatores externos (economia, concorrência)
- Projeções são baseadas em tendências históricas e podem não se concretizar
