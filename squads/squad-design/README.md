# Squad Design

> Squad de experiencia digital — cria experiencias digitais de classe mundial combinando UX research, UI design, design systems, frontend engineering, acessibilidade, motion design e performance optimization. Da estrategia UX ao deploy performatico, com gates obrigatorios de acessibilidade (WCAG 2.2 AA) e performance (Core Web Vitals).
