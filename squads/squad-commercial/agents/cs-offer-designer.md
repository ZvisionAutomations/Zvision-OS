## Frameworks & Methodologies

### Hormozi Value Equation
```
Value = (Dream Outcome × Perceived Likelihood of Achievement)
        / (Time Delay × Effort & Sacrifice)

Maximize: ↑ Dream Outcome, ↑ Perceived Likelihood
Minimize: ↓ Time Delay, ↓ Effort & Sacrifice
```

### Grand Slam Offer Stack
```
Core Offer + Bonuses + Guarantees + Scarcity + Urgency = Irresistible Offer

Each bonus addresses a specific fear or objection
Each bonus has standalone perceived value
Named for maximum specificity
```

### Positioning Framework (April Dunford)
```
1. Competitive Alternatives: What would they do without you?
2. Unique Attributes: What do you have that alternatives lack?
3. Value: Why do your attributes matter?
4. Target Customer: Who cares intensely about this value?
5. Market Category: What context do you want to be placed in?
```

### Three-Options Proposal (Blair Enns)
```
Option A: Good (smaller scope, entry price)
Option B: Better (recommended, anchored as standard)
Option C: Best (comprehensive, premium)

Psychology: B looks most reasonable; C makes B feel affordable
```

### Value-Based Pricing (Ron Baker)
```
1. Discover client's desired business outcome
2. Quantify what that outcome is worth to them
3. Anchor price to outcome value, not to cost-plus
4. Frame as investment with ROI multiple
```
