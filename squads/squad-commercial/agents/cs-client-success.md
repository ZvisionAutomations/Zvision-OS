## Frameworks & Methodologies

### Desired Outcome Framework (Lincoln Murphy)
```
Desired Outcome = Required Outcome + Appropriate Experience

Required Outcome: The functional goal
Appropriate Experience: How they need to achieve it
Churn happens when gap > patience threshold
```

### Customer Health Score (Gainsight)
```
Health Score = Weighted composite:
  - Product Usage / Engagement (25-40%)
  - Adoption of Key Features (15-25%)
  - Support Ticket Volume/Sentiment (10-15%)
  - NPS / Satisfaction Score (10-15%)
  - Contract Status / Renewal Date (10-20%)
  - Business Outcomes Achieved (15-20%)

Green (80-100): Healthy, expansion candidate
Yellow (50-79): At risk, proactive intervention
Red (0-49): Critical risk, save play required
```

### CS Maturity Model
| Level | Description |

| 1 - Reactive | Respond to tickets, no proactive CS |
| 2 - Informed | Basic health tracking, some proactive outreach |

| 5 - Transformative | CS drives product roadmap and revenue strategy |

### Net Promoter Score (Fred Reichheld)
```
NPS = % Promoters (9-10) - % Detractors (0-6)
Passives (7-8) excluded

Agency benchmark: 30-50 typical, 60+ world-class
```

### Trust Equation (David Maister)
```
Trust = (Credibility + Reliability + Intimacy) / Self-Orientation

Self-Orientation is the denominator — most powerful variable
```
