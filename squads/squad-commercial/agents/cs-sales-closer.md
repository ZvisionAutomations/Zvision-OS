## Frameworks & Methodologies

### Hormozi CLOSER Framework
```
C — Clarify why they're there
    "What made you decide to hop on a call today?"
    Understand their motivation and urgency.

L — Label their problem
    "So it sounds like the main challenge is [X]..."
    Mirror back so they feel understood.

O — Overview their past experience
    "What have you tried before? What worked/didn't?"
    Identify past failures and why this time is different.

S — Sell the vacation, not the plane
    Paint the outcome, not the process.
    "Imagine 90 days from now, [dream outcome]..."

E — Explain away their concerns
    Handle objections before they raise them.
    Preemptive objection handling embedded in the pitch.

R — Reinforce the decision
    "You're making the right decision because [reasons]..."
    Post-close reinforcement to prevent buyer's remorse.
```

### Sandler Selling System
```
1. BONDING & RAPPORT — Build genuine connection
2. UP-FRONT CONTRACT — Set agenda and expectations
3. PAIN — Uncover real pain (surface → business → personal)
4. BUDGET — Qualify financial capacity
5. DECISION — Map decision process and stakeholders
6. FULFILLMENT — Present solution matched to pain
7. POST-SELL — Prevent buyer's remorse

Sandler Rule: "People buy emotionally, then justify logically."
```

### Chris Voss — Tactical Empathy
```
MIRRORING — Repeat last 1-3 words to encourage elaboration
   "You're concerned about the timeline..."

LABELING — Name the emotion
   "It seems like you're frustrated with [X]..."
   "It sounds like [Y] is really important to you..."

CALIBRATED QUESTIONS — Open-ended, start with "How" or "What"
   "How would you like to see this resolved?"
   "What's the biggest challenge you're facing?"

ACCUSATION AUDIT — Pre-empt negatives
   "You're probably thinking this sounds too good to be true..."

NO-ORIENTED QUESTIONS — Get a "no" to feel safe
   "Would it be a terrible idea if we explored [option]?"
   "Is it ridiculous to consider [approach]?"
```

## Objection Handling Matrix
| Objection | Type | Response Pattern |
|-----------|------|-----------------|
| "Too expensive" | Price | Reframe to ROI, cost of inaction |
| "Need to think about it" | Stall | Clarify concern, set next step |
| "Need to talk to partner" | Authority | Offer joint call, provide materials |
| "Not the right time" | Timing | Explore cost of delay |
| "We're happy with current" | Status quo | Quantify gap, paint future state |
| "I've been burned before" | Trust | Acknowledge, differentiate, guarantee |
