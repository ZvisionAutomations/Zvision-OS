# Hormozi Business Frameworks Reference

Comprehensive reference for Alex Hormozi's frameworks from $100M Offers, $100M Leads, and Gym Launch.

## Value Equation ($100M Offers)
```
Value = (Dream Outcome x Perceived Likelihood of Achievement) / (Time Delay x Effort & Sacrifice)
```
- Maximize: Dream Outcome (vivid picture of life AFTER), Perceived Likelihood (social proof, guarantees)
- Minimize: Time Delay (quick wins in 24-48h), Effort & Sacrifice (done-for-you, templates, automation)

## Grand Slam Offer Creation (5 Steps)
1. Identify dream outcome
2. List EVERY obstacle between customer and dream outcome
3. Create solution for EVERY problem
4. Turn solutions into delivery vehicles (1-on-1, small group, one-to-many, DIY)
5. Trim and stack — low-cost/high-value items become bonuses

## Bonus Stacking Rules
- Each bonus solves the NEXT problem after core offer
- Each has standalone perceived value with dollar amount
- Stack: Core Offer + Bonus 1 (objection A) + Bonus 2 (objection B) + Bonus 3 (accelerator)

## Guarantee Engineering
| Type | Description | Best For |
|------|-------------|----------|
| Unconditional | Full money-back, no questions | High-confidence delivery |
| Conditional | Money-back IF conditions met | Results-based services |
| Anti-guarantee | "If it doesn't work, I'll work for free until it does" | High-ticket consulting |
| Performance | Refund + extra payment if fail | Extreme confidence |

## $100M Leads Framework
- Core 4 ways to get customers: Warm outreach, Cold outreach, Content (free), Paid ads
- Lead Magnet: solve a narrow problem for free to generate leads
- Each method: daily actions → leads → customers → revenue

## CLOSER Framework (Sales)
C = Clarify why they're here
L = Label the problem
O = Overview past attempts
S = Sell the vacation (not the plane ride)
E = Explain away concerns
R = Reinforce decision

## Scaling Framework
- Lead getting → lead nurture → sales → fulfillment → retention → referral
- Constraint theory: find the bottleneck, fix it, move to next
- Hiring: A-players only, fire fast, train constantly
