# Offer Design Frameworks

## Hormozi Value Equation

### The Formula
```
                Dream Outcome × Perceived Likelihood of Achievement
VALUE = ─────────────────────────────────────────────────────────────
                Time Delay × Effort & Sacrifice Required

To increase perceived value:
  ↑ Dream Outcome: Make the outcome more desirable
  ↑ Perceived Likelihood: Increase confidence it will work
  ↓ Time Delay: Reduce time to first result
  ↓ Effort & Sacrifice: Make it easier for the client
```

### Scoring Model
| Dimension | Score (1-10) | Weight | Weighted Score |
|-----------|-------------|--------|---------------|
| Dream Outcome | | 30% | |
| Perceived Likelihood | | 30% | |
| Time Delay (inverse) | | 20% | |
| Effort & Sacrifice (inverse) | | 20% | |
| **Total Value Score** | | | /10 |

### Agency Application
```
BEFORE (Generic Agency Offer):
  "We provide digital marketing services"
  Dream Outcome: 4/10 (vague)
  Likelihood: 3/10 (no proof)
  Time Delay: 3/10 (results in 6-12 months?)
  Effort: 4/10 (lots of meetings, approvals)
  VALUE SCORE: ~3.5/10

AFTER (Hormozi-Optimized Offer):
  "We guarantee 50 qualified leads in 90 days for
  B2B SaaS companies, or we work free until we hit it"
  Dream Outcome: 8/10 (specific, measurable)
  Likelihood: 8/10 (guarantee removes risk)
  Time Delay: 7/10 (90-day timeline)
  Effort: 7/10 (done-for-you)
  VALUE SCORE: ~7.5/10
```

## Grand Slam Offer Stack (Alex Hormozi)

### Components
```
1. CORE OFFER
   The main deliverable / service
   → What they're primarily paying for

2. BONUSES (Value Stacking)
   Additional value that increases perceived worth
   → Each bonus solves a related problem
   → Each has standalone value but costs little to deliver
   → "But wait, you also get..."

3. GUARANTEE (Risk Reversal)
   Remove the risk of saying yes
   Types:
     Unconditional: "Money back, no questions asked"
     Conditional: "If we don't hit X, you don't pay" (performance-based)
     Anti-guarantee: "We guarantee nothing — but look at these results"
     Implied: "We've done this for 47 companies like yours"

4. SCARCITY (Real, Not Fake)
   Limit availability based on genuine constraints
   → "We only onboard 3 new clients per month" (capacity-based)
   → "This pricing is available for Q1 contracts only" (time-based)
   → Must be TRUE — fake scarcity destroys trust

5. URGENCY
   Reason to act now vs later
   → Cost of delay (quantified)
   → Market window closing
   → Competitive advantage of early action
```

### Value Stack Example (Agency)
```
CORE OFFER: Full-funnel marketing program
  Value: R$15K/month

BONUS 1: Custom ICP research report
  Standalone value: R$5K
  Cost to deliver: R$500 (already do this in discovery)

BONUS 2: Competitive intelligence monthly brief
  Standalone value: R$3K
  Cost to deliver: R$300 (automated monitoring)

BONUS 3: Quarterly executive presentation deck
  Standalone value: R$2K
  Cost to deliver: R$200 (template-based)

GUARANTEE: "If we don't generate 30 qualified leads in 90 days,
we extend our engagement free until we do."

TOTAL PERCEIVED VALUE: R$25K/month
ACTUAL PRICE: R$15K/month
PERCEIVED VALUE RATIO: 1.67x
```

## Positioning Framework (April Dunford)

### Five Components of Positioning
```
1. COMPETITIVE ALTERNATIVES
   "If we didn't exist, what would they use?"
   → Not just direct competitors
   → Includes: do nothing, hire internally, use spreadsheets, etc.

2. UNIQUE ATTRIBUTES
   "What do we have that alternatives don't?"
   → Must be provably different (not just "better")
   → Features, capabilities, process, team, approach

3. VALUE
   "What value do those unique attributes provide?"
   → Translate attributes into outcomes
   → Each attribute → specific customer benefit

4. TARGET CUSTOMER
   "Who cares most about that value?"
   → The segment where your differentiation matters most
   → Be specific: industry, size, stage, trigger

5. MARKET CATEGORY
   "What market do customers put you in?"
   → Frame sets expectations
   → Choose: existing category, adjacent category, or create new
```

### Positioning Canvas (Agency)
```
COMPETITIVE ALTERNATIVES:
  □ Large holding-group agencies (WPP, Publicis, etc.)
  □ Boutique specialist agencies
  □ In-house marketing team
  □ Freelance network / marketplace
  □ Do nothing / status quo

UNIQUE ATTRIBUTES:
  □ Senior-level talent on every account (not bait-and-switch)
  □ Methodology-driven (not just creative instinct)
  □ Full-stack capability (strategy through execution)
  □ Performance guarantees (skin in the game)
  □ [Agency-specific: proprietary process, tech, data, etc.]

VALUE (Attribute → Benefit):
  Senior talent → Faster decisions, less rework, strategic partnership
  Methodology → Predictable results, transferable knowledge
  Full-stack → One partner, no coordination overhead
  Guarantees → Risk reduction, confidence in investment

TARGET CUSTOMER:
  "Mid-market B2B companies (R$10-100M revenue) going through
  digital transformation with internal marketing teams of 3-10
  who need strategic + execution capacity they can't build fast enough"

MARKET CATEGORY:
  "Strategic digital partner" (not "agency", not "vendor")
```

## Three-Options Proposal Framework (Blair Enns)

### The Psychology
```
One option = take it or leave it (50% close rate)
Two options = which one? or neither? (65% close rate)
Three options = WHICH one? (80%+ close rate)

The shift: From "should I buy?" to "which should I buy?"
```

### Structure
```
OPTION 1: GOOD (The Foot in the Door)
  Scope: Minimum viable engagement
  Solves: Core problem only
  Price: Lowest (anchor for value comparison)
  Purpose: Shows you're flexible, creates baseline

OPTION 2: BETTER (The Recommended Sweet Spot)
  Scope: Complete solution addressing all discovered needs
  Solves: Core + adjacent problems
  Price: ~2x Option 1
  Purpose: The one you WANT them to choose (mark as "Recommended")

OPTION 3: BEST (The Aspirational Anchor)
  Scope: Transformational engagement
  Solves: Everything + strategic partnership
  Price: ~3-4x Option 1
  Purpose: Anchoring effect, makes Option 2 feel reasonable

PRESENTATION ORDER: 3 → 1 → 2
  Show the most expensive first (anchoring)
  Then the cheapest (creates contrast)
  Then the recommended middle (feels like the smart choice)
```

### Pricing Psychology
```
Example:
  Option 1: R$8K/month → "Focus" (one channel, tactical)
  Option 2: R$15K/month → "Growth" (multi-channel, strategic) ← RECOMMENDED
  Option 3: R$35K/month → "Transformation" (full-stack, embedded team)

Key rules:
  1. All three must GENUINELY solve the core problem
  2. Higher options add VALUE, not just volume
  3. Option 1 cannot be a "throwaway" — must be viable
  4. Option 3 must be defensible, not obviously overpriced
  5. Price gaps should be intentional, not linear
```

## Value-Based Pricing (Ron Baker / Blair Enns)

### Principles
```
1. Price the client, not the deliverable
   → Same work can be worth R$10K to one company and R$100K to another
   → Price to the VALUE they receive, not the COST to you

2. Never use hourly rates
   → Hourly billing punishes efficiency
   → Aligns incentives incorrectly (more hours = more revenue)
   → Frame as investment in outcomes, not purchase of time

3. Quantify the value before pricing
   → "This project will generate R$500K in additional revenue"
   → Price at 10-20% of expected value = R$50-100K
   → ROI is still 5-10x, making the investment obvious

4. Present price AFTER value is established
   → Never lead with price
   → Build the value story through the document
   → By the time they see the price, the ROI is already clear
```

### Value Pricing Formula
```
Price = (Client's Expected Value × Value Capture %) ÷ Risk Factor

Where:
  Expected Value = quantified business impact
  Value Capture = 10-25% of value created
  Risk Factor = 1.0 (guaranteed) to 2.0 (uncertain)

Example:
  Expected Value: R$1M additional revenue from campaign
  Value Capture: 15%
  Risk Factor: 1.2 (moderate confidence)
  Price = (R$1M × 0.15) / 1.2 = R$125K
```

## Productized Services (Patrick McKenzie / Brennan Dunn)

### Model
```
CUSTOM SERVICE → PRODUCTIZED SERVICE
  Custom: Unique scope per client, custom pricing, unpredictable delivery
  Productized: Fixed scope, fixed price, standardized delivery

Benefits:
  ✓ Easier to sell (clear deliverable, clear price)
  ✓ Easier to deliver (repeatable process)
  ✓ Higher margins (efficiency through standardization)
  ✓ Scalable (can train team on standard process)

Framework:
  1. Identify your most repeated engagement type
  2. Define fixed scope (what's IN, what's OUT)
  3. Set fixed price (based on value, not hours)
  4. Create standard process (SOP for delivery)
  5. Package and name it (make it feel like a product)
```

### Agency Productization Examples
| Service | Productized Version | Fixed Price | Delivery |
|---------|-------------------|-------------|----------|
| Brand strategy | "Brand Sprint" — 5-day brand workshop | R$25K | 1 week |
| Website design | "Site in 30" — website in 30 days | R$40K | 30 days |
| Content strategy | "Content Blueprint" — 90-day content plan | R$15K | 2 weeks |
| SEO audit | "SEO X-Ray" — full technical + content audit | R$8K | 10 days |
| Marketing strategy | "Growth Map" — full-funnel strategy | R$20K | 3 weeks |
