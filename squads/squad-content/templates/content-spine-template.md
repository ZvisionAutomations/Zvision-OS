## Espinha Dorsal

### TESE
> {Afirmacao central em 1 frase. Deve ser clara, defendivel e provocativa.}

**Tipo de tese**: {contraintuitiva / observacional / predictiva / prescritiva}

### MECANISMO
> {Como/por que a tese funciona. Deve ser especifico e ensinavel.}

**Formato do mecanismo**: {framework / principio / analogia / modelo mental / processo}

### PROVA
> {Evidencia concreta que sustenta tese + mecanismo.}

**Tipo de prova**: {dado estatistico / case real / exemplo do setor / experiencia pessoal / analogia}
**Fonte**: {referencia}

### DIRECAO
> {O que a audiencia faz depois de consumir.}

**Tipo de direcao**: {reflexao / acao imediata / engajamento / aprofundamento / conversao}

## Validacao

| Criterio | Pass? |
|----------|:-----:|
| TESE → MECANISMO: conexao logica? | ☐ |
| MECANISMO sustentado pela PROVA? | ☐ |
| PROVA direciona para DIRECAO? | ☐ |
| Cada componente e ESPECIFICO (nao generico)? | ☐ |
| Alinhado com pilar e brand voice? | ☐ |
