# Signal Detection Methods

> Framework de deteccao de sinais para identificar oportunidades de conteudo antes da concorrencia.

---

## O Que Sao Sinais?

Sinais sao indicadores de mudanca no ambiente que representam oportunidades para conteudo. Podem ser: trends emergentes, noticias do setor, movimentos de concorrentes, mudancas de comportamento da audiencia, eventos culturais, ou dados de performance que indicam novos interesses.

---

## Tipos de Sinais

| Tipo | Fonte | Velocidade | Exemplo |
|------|-------|-----------|---------|
| **Trend Signal** | Redes sociais, Google Trends | Alta (horas-dias) | Hashtag explodindo, meme viral |
| **News Signal** | Portais, newsletters setoriais | Media (dias) | Regulacao nova, aquisicao, lancamento |
| **Cultural Signal** | Sociedade, comportamento | Lenta (semanas-meses) | Mudanca de habito, novo valor social |
| **Competitive Signal** | Concorrentes | Media (dias-semanas) | Concorrente publica conteudo de alto engajamento |
| **Audience Signal** | DMs, comentarios, reviews | Variavel | Pergunta recorrente, reclamacao, elogio |
| **Performance Signal** | Analytics proprios | Continua | Topico com engagement acima da media |
| **UGC Signal** | Comunidade | Variavel | Hashtag da marca usada organicamente |

## Fontes por Categoria

### Fontes Primarias (Monitoramento Ativo)
- Google Trends (tendencias de busca)
- Social listening (mentions, hashtags)
- Plataformas de trends (Exploding Topics, SparkToro)
- Newsletters setoriais (curadoria humana)
- Comentarios e DMs proprios

### Fontes Secundarias (Monitoramento Passivo)
- Portais de noticias do setor
- Podcasts de referencia
- Foruns e comunidades (Reddit, Quora, Discord)
- Publicacoes de influencers do setor
- Relatorios de mercado (Statista, eMarketer, etc.)

### Fontes Competitivas
- Perfis de concorrentes (publicacoes, frequencia, formatos)
- SEO competitivo (keywords rankeadas pelos concorrentes)
- Share of voice (participacao de voz no setor)

---

## Processo de Deteccao

```
1. SCAN → Varredura automatica diaria de todas as fontes
2. FILTER → Eliminar ruido: sinal conecta com pilares? E relevante?
3. CLASSIFY → Classificar temperatura (HOT/WARM/COLD) via SPV
4. MAP → Mapear sinal para pilar editorial
5. ROUTE → Encaminhar: HOT → alerta, WARM → briefing, COLD → arquivo
```

## Calibracao por Setor

| Setor | Velocidade | Threshold SPV para HOT | Fontes Prioritarias |
|-------|-----------|----------------------|---------------------|
| Tech/Startups | Rapido | >= 3.5 | Twitter/X, TechCrunch, Product Hunt |
| Moda/Beleza | Rapido | >= 3.5 | Instagram, TikTok, Vogue |
| B2B Enterprise | Lento | >= 4.0 | LinkedIn, Forbes, Harvard Business Review |
| Saude/Bem-estar | Medio | >= 4.0 | PubMed, Instagram, Podcasts |
| Educacao | Medio | >= 3.5 | Twitter/X, Reddit, blogs educacionais |
| Financas | Medio-Rapido | >= 4.0 | Bloomberg, LinkedIn, reguladores |
