# Narrative Structures Library

> Biblioteca de estruturas narrativas aplicaveis a conteudo de marca.

---

## 1. Arco Classico (3 Atos)

```
Ato 1: Setup (25%) → Ato 2: Confronto (50%) → Ato 3: Resolucao (25%)
```

- **Setup**: Apresentar personagem + mundo + desafio
- **Confronto**: Tensao crescente, obstaculos, tentativas
- **Resolucao**: Climax + transformacao + novo equilibrio

**Quando usar**: Cases, stories de transformacao, conteudo longo.

---

## 2. Hook-Value-CTA (Formato Social)

```
Hook (10%) → Valor (80%) → CTA (10%)
```

- **Hook**: Parar o scroll em 1.5-3 segundos
- **Valor**: Entregar insight/framework/acao
- **CTA**: Direcionar para proxima acao

**Quando usar**: Todo conteudo social. Framework universal.

---

## 3. AIDA (Attention-Interest-Desire-Action)

| Etapa | Funcao | Em Conteudo |
|-------|--------|-------------|
| Attention | Capturar atencao | Hook, dado, pergunta |
| Interest | Manter interesse | Contexto, historia, tensao |
| Desire | Criar desejo | Beneficio, prova social, transformacao |
| Action | Provocar acao | CTA claro e especifico |

**Quando usar**: Conteudo de conversao (FUNDO de funil).

---

## 4. PAS (Problem-Agitate-Solve)

| Etapa | Funcao |
|-------|--------|
| **Problem** | Nomear a dor especifica da audiencia |
| **Agitate** | Amplificar a dor — mostrar consequencias de nao resolver |
| **Solve** | Apresentar a solucao (seu produto/framework/conselho) |

**Quando usar**: Conteudo comercial elegante. Selling sem "vender".

---

## 5. Before-After-Bridge

| Etapa | Funcao |
|-------|--------|
| **Before** | O mundo ANTES (dor, frustacao, problema) |
| **After** | O mundo DEPOIS (resultado, transformacao) |
| **Bridge** | COMO chegar la (produto, framework, acao) |

**Quando usar**: Lancamentos, cases, depoimentos.

---

## 6. Star-Chain-Hook (Frank Dignan)

| Etapa | Funcao |
|-------|--------|
| **Star** | Ideia ou personagem central que brilha |
| **Chain** | Cadeia de fatos, beneficios, razoes |
| **Hook** | CTA irresistivel que fecha |

**Quando usar**: Posts de opiniao, thought leadership.

---

## 7. Inverted Pyramid (Jornalismo)

```
Lead (mais importante) → Corpo (desenvolvimento) → Base (detalhes secundarios)
```

**Quando usar**: Noticias, alertas, atualizacoes. Valor na primeira frase.

---

## 8. The Gap (Nancy Duarte)

```
"O Que E" → "O Que Poderia Ser" → Alternancia → Visao Final
```

**Principio**: Criar tensao entre realidade atual e possibilidade futura, alternando entre os dois ate convergir na visao transformadora.

**Quando usar**: Apresentacoes, manifestos, conteudo inspiracional.
