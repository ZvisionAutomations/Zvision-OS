# C-Level Squad

C-suite virtual com 6 agentes de lideranca executiva.

## Inicio Rapido

```
@vision-chief        # Ativar o CEO
*diagnose            # Triagem do seu desafio executivo
*strategic-planning  # Planejamento estrategico completo
*board-presentation  # Preparar apresentacao para o conselho
```

## Agentes (6)

- **vision-chief** — CEO / Orquestrador
- **coo-orchestrator** — COO / Operacoes
- **cmo-architect** — CMO / Marketing
- **cto-architect** — CTO / Tecnologia
- **cio-engineer** — CIO / Infraestrutura
- **caio-architect** — CAIO / Estrategia de IA

## Componentes

- **6 agentes**, **7 tasks**, **2 workflows**, **1 checklist**

## Requisitos

- AIOS >= 4.0.0
