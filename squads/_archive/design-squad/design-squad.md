# Design Squad

Squad de design operations — 3 experts + 4 especialistas + 1 orquestrador.

## Inicio Rapido

```
@design-chief           # Ativar o orquestrador
*diagnose               # Triagem do seu desafio de design
*design-system-creation # Criar um design system
*feature-design         # Projetar uma feature ponta a ponta
```

## Agentes (8)

- **design-chief** — Orquestrador
- **brad-frost** — Metodologia Atomic Design
- **dan-mall** — Design que escala
- **dave-malouf** — Lideranca de DesignOps
- **ux-designer** — Pesquisa & design de UX
- **design-system-architect** — Bibliotecas de componentes
- **visual-generator** — Assets visuais
- **ui-engineer** — Producao de codigo UI
