# Cybersecurity Squad

15 agentes para operacoes de seguranca ofensiva e defensiva.

## Inicio Rapido

```
@cyber-chief         # Ativar o orquestrador
*diagnose            # Triagem do seu desafio de seguranca
*pentest-engagement  # Executar um teste de penetracao
*incident-response   # Tratar um incidente de seguranca
```
