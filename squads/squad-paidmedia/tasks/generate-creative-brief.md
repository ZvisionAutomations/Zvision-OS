# Task: Generate Creative Brief

## Metadata
- **Agent:** creative-strategist (Canvas)
- **Squad:** squad-paidmedia
- **Complexity:** MEDIUM

## Objetivo
Gerar brief completo de criacao para campanha de ads com audience, messaging, visual direction e specs por plataforma.

## Steps

1. **Definir audience e context**
   - Quem e o target? (demographics, psychographics, pain points)
   - Onde estao no funil? (awareness, consideration, conversion)
   - Que plataforma? (Meta, Google, LinkedIn, TikTok)
   - Qual o objetivo? (clicks, leads, sales)

2. **Definir messaging hierarchy**
   - Primary message: a unica coisa que a pessoa deve lembrar
   - Support points: 2-3 razoes para acreditar
   - CTA: acao desejada, clara e urgente
   - Tone of voice: alinhado com brand guidelines

3. **Definir visual direction**
   - Mood: que emocao/sensacao o criativo deve evocar?
   - Color palette: cores primarias e acento
   - Typography style: bold/modern/classic/playful
   - Imagery: UGC, lifestyle, product-focused, abstract
   - Reference: 2-3 exemplos de referencia visual

4. **Specs por plataforma**
   | Plataforma | Formato | Tamanho | Duracao (video) |
   |-----------|---------|---------|-----------------|
   | Meta Feed | 1:1, 4:5 | 1080x1080, 1080x1350 | 15-30s |
   | Meta Stories/Reels | 9:16 | 1080x1920 | 15s |
   | Google Display | Multiple | 300x250, 728x90, etc | - |
   | YouTube | 16:9 | 1920x1080 | 15/30/60s |
   | LinkedIn | 1:1, 1.91:1 | 1080x1080, 1200x627 | 30-90s |

5. **Definir variacoes para teste**
   - Hook variations (min 3)
   - CTA variations (min 2)
   - Format variations (image + video + carousel)

## Output
Creative brief usando creative-brief-template.md.

## Quality Gates
- [ ] Audience claramente definida
- [ ] Messaging hierarchy completa
- [ ] Visual direction com referencias
- [ ] Specs por plataforma corretos
- [ ] Variacoes para teste definidas

## Quando Usar
- Inicio de campanha nova
- Refresh de criativos existentes
