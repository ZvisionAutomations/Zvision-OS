# Task: Track GMB Performance Trends

## Metadata
- **Agent:** campaign-analyst (Pulse)
- **Squad:** squad-paidmedia
- **Complexity:** LOW

## Objetivo
Rastrear tendencias de performance GMB ao longo do tempo.

## Steps

1. **Metricas mensais** - Searches (direct, discovery, branded). Views (maps, search). Actions (calls, website, directions)
2. **Trends MoM** - Crescimento/queda por metrica. Sazonalidade
3. **Photo views** - Views de fotos do business vs concorrentes
4. **Correlacao com campanhas** - Impacto de local ads no GMB performance. Impacto de posts no engagement

## Output
GMB performance trends report com metricas, trends e correlacoes.

## Quality Gates
- [ ] Metricas de 3+ meses para trend
- [ ] MoM comparativo

## Quando Usar
- Report mensal de local
- Review trimestral de estrategia local
