# Task: Analyze Local Keywords

## Metadata
- **Agent:** campaign-analyst (Pulse)
- **Squad:** squad-paidmedia
- **Complexity:** LOW

## Objetivo
Analisar keywords locais e oportunidades para local ads e SEO.

## Steps

1. **Identificar keyword patterns locais** - "[servico] + [cidade]", "[servico] perto de mim", "[servico] + [bairro]"
2. **Volume e competitividade** - Volume de busca por keyword. CPC estimado (para paid). Dificuldade organica
3. **Cross-reference com campanhas** - Keywords locais ja cobertas por paid? Keywords com oportunidade organica?
4. **Recomendacoes** - Keywords para local Search campaigns. Keywords para conteudo local (SEO). Long-tail local com baixa concorrencia

## Output
Local keyword analysis com volume, competitividade e recomendacoes.

## Quality Gates
- [ ] Keywords locais com volume estimado
- [ ] Cross-reference com paid existente

## Quando Usar
- Setup de campanhas locais
- Expansao de presenca local
