# Task: Write Video Ad Script

## Metadata
- **Agent:** creative-strategist (Canvas)
- **Squad:** squad-paidmedia
- **Complexity:** MEDIUM

## Objetivo
Escrever script de video ad com hook poderoso, estrutura comprovada e otimizacao para sound-off.

## Steps

1. **Definir estrutura do video**
   - **Hook (0-3s):** A parte mais critica. 65% do impacto. Deve parar o scroll.
   - **Problem (3-10s):** Agitar a dor/desejo do target. Empatia.
   - **Solution (10-25s):** Apresentar o produto/servico como solucao. Proof points.
   - **CTA (ultimos 3-5s):** Acao clara, urgente, com incentivo se possivel.

2. **Criar 3+ hook variations usando formulas**
   - **Question:** "Voce sabia que [estatistica surpreendente]?"
   - **Statistic:** "[Numero impactante] + [contexto relevante]"
   - **Bold Statement:** "[Afirmacao controversa ou contra-intuitiva]"
   - **Demonstration:** [Mostrar resultado/antes-depois nos primeiros 3s]
   - **Testimonial:** "[Depoimento impactante de cliente]"
   - **Pattern Interrupt:** [Visual/audio inesperado que quebra o pattern]

3. **Escrever script completo**
   - Coluna VISUAL: o que aparece na tela
   - Coluna AUDIO: narracao ou dialogo
   - Coluna TEXT OVERLAY: texto que aparece (para sound-off)
   - Coluna TIMING: duracao de cada segmento
   - REGRA: text overlay deve contar a historia completa sem audio

4. **Adaptar por plataforma**
   - YouTube pre-roll: hook em 5s (skipable), CTA forte
   - Meta Feed: 15-30s max, 4:5 ou 1:1, sound-off first
   - Meta Reels/Stories: 15s max, 9:16, nativo e informal
   - TikTok: 9-15s, UGC style, trending sounds
   - LinkedIn: 30-90s, profissional, data-driven

5. **Sound-off optimization**
   - Todo texto falado tem overlay equivalente
   - Visual storytelling que funciona sem audio
   - Captions: white text, dark background, large font
   - Musica de fundo: opcional mas ajuda com sound-on

## Output
Script completo com visual, audio, text overlay e timing por secao.

## Quality Gates
- [ ] Hook nos primeiros 3 segundos
- [ ] 3+ hook variations escritas
- [ ] Text overlay conta a historia sem audio
- [ ] Duracao dentro do spec da plataforma
- [ ] CTA claro e urgente

## Quando Usar
- Producao de video ads para qualquer plataforma
- Refresh de video ads existentes
- Teste de hooks
