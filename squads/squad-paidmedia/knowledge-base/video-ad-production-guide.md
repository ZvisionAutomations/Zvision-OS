# Video Ad Production Guide

## Universal Video Ad Structure

### The 4-Act Framework
```
ACT 1: HOOK (0-3 segundos)
├── Objetivo: parar o scroll
├── 65% da performance vem daqui
├── Text overlay OBRIGATORIO
└── Tecnicas: question, stat, bold statement, demo, testimonial, pattern interrupt

ACT 2: PROBLEM (3-10 segundos)
├── Objetivo: gerar identificacao
├── Agitar a dor/frustacao
├── "Voce ja se sentiu [dor]?"
└── Mostrar consequencia de nao resolver

ACT 3: SOLUTION (10-25 segundos)
├── Objetivo: apresentar o produto/servico
├── Beneficio > Feature (sempre)
├── Social proof (numeros, testimonials, logos)
├── Demonstracao visual do produto em acao
└── 2-3 proof points max

ACT 4: CTA (ultimos 3-5 segundos)
├── Objetivo: conversao
├── Acao clara: "Clique agora", "Comece gratis"
├── Urgencia: "Oferta por tempo limitado"
├── Incentivo: "Frete gratis", "30 dias gratis"
└── Visual: botao ou seta apontando para CTA
```

## Platform-Specific Structures

### Meta Feed (15-30 segundos)
```
Aspecto: 4:5 (1080x1350) ou 1:1 (1080x1080)
Sound-off first: text overlay conta historia sozinha
Captions: white text, dark semi-transparent background
Hook: 0-3s (CRITICO — decide se continua ou scrolla)
Body: 3-20s
CTA: ultimos 5s + CTA card Meta
Tip: 15s e sweet spot para engagement
```

### Meta Reels/Stories (9-15 segundos)
```
Aspecto: 9:16 (1080x1920)
Estilo: nativo, informal, UGC-like
Hook: 0-2s (mais rapido que Feed)
Body: 2-10s
CTA: ultimos 3s
Tip: parecer conteudo organico, nao ad
Tip: usar trending audio (Reels)
Safe zone: texto central, evitar bordas (UI overlay)
```

### YouTube Pre-roll (15/30/60 segundos)
```
Aspecto: 16:9 (1920x1080)
Sound ON (YouTube = audio environment)
Skippable: hook em 5s (antes do botao Skip)
Non-skippable: 15s completos
Hook: 0-5s (mostrar logo + valor ANTES do skip)
Body: 5-25s
CTA: ultimos 5s + end card + CTA overlay
Tip: CTA a partir do segundo 5 (nao esperar o final)
```

### TikTok (9-15 segundos)
```
Aspecto: 9:16 (1080x1920)
Estilo: UGC first, autentico, informal
Hook: 0-2s (TikTok users scrollam mais rapido)
Body: 2-10s
CTA: ultimos 3s
Tip: usar trending sounds e formats
Tip: "text on screen" no estilo TikTok nativo
Tip: parecer creator, nao marca
```

### LinkedIn (30-90 segundos)
```
Aspecto: 1:1 (1080x1080) ou 16:9 (1920x1080)
Estilo: profissional, data-driven, thought leadership
Hook: 0-5s (profissional, nao sensacionalista)
Body: 5-70s (pode ser mais longo, audience profissional)
CTA: ultimos 10s
Tip: stats e dados performam bem
Tip: face-to-camera para credibilidade
Tip: subtitles sempre (muitos assistem no escritorio sem som)
```

## Sound-Off Optimization

### Regra: 85% dos usuarios assistem sem som

### Checklist Sound-Off
- [ ] Text overlay conta a historia completa sem audio
- [ ] Captions em todo texto falado
- [ ] Estilo de caption: legivel em mobile (min 24px)
- [ ] Contraste: texto claro em fundo escuro semi-transparente
- [ ] Elementos visuais comunicam emocao (nao depender de musica)
- [ ] CTA visual (seta, botao, highlight)

### Caption Styles
| Estilo | Quando Usar |
|--------|------------|
| Bottom center bar | Default para Feed |
| Word-by-word highlight | TikTok/Reels style |
| Full screen text | Bold statements, hooks |
| Burned-in subtitles | YouTube, LinkedIn |

## Text Overlay Strategy

### Hierarquia de Texto no Video
1. **Hook text (0-3s):** GRANDE, bold, central — a mensagem mais importante
2. **Supporting text (3-15s):** medio, complementa visuals
3. **CTA text (final):** GRANDE, contrastante, urgente

### Rules
- Max 7 palavras por overlay (leitura em 2s)
- Font: sans-serif, bold, min 24px em mobile
- Animacao: simple fade/slide in (nao distrair)
- Posicao: centro ou terco superior (zona de atencao)
- Cor: contraste alto com background (outline ou shadow se necessario)

## Messaging Hierarchy Framework

### Para cada video ad, definir:
1. **Primary Message (1 frase):** O que o viewer DEVE lembrar
2. **Support Point 1:** Prova ou beneficio #1
3. **Support Point 2:** Prova ou beneficio #2
4. **CTA:** Acao clara + incentivo

### Nao cometer:
- Tentar comunicar 5 beneficios em 15 segundos
- Comecar com logo da marca (ninguem liga)
- Usar jargao tecnico em cold audiences
- CTA generico ("Saiba mais" sem especificidade)

## Visual Direction Matrix

### Por Objetivo
| Objetivo | Visual Style | Tone | Ritmo |
|----------|-------------|------|-------|
| Awareness | Bold, colorido, impactante | Inspiracional | Rapido |
| Consideration | Demonstracao, social proof | Educativo | Medio |
| Conversion | Produto em uso, preco, oferta | Urgente | Rapido |

### Por Audience Temperature
| Audience | Visual | Messaging Focus |
|----------|--------|----------------|
| Cold | Emocional, pain-driven | Problema + Big promise |
| Warm | Racional, proof-driven | Beneficio + Social proof |
| Hot | Direto, offer-driven | Oferta + Urgencia + CTA |

## Production Decision: Polished vs UGC

| Fator | Polished | UGC |
|-------|---------|-----|
| Cold audiences | ❌ Menos trust | ✅ Mais autentico |
| Warm audiences | ✅ Brand building | ✅ Funciona |
| Hot audiences | ✅ Profissionalismo | ❌ Pode parecer casual demais |
| Meta Feed | ❌ Parece ad (scroll past) | ✅ Parece organico |
| YouTube | ✅ Production value esperada | ❌ Pode parecer amador |
| TikTok | ❌ Nao-nativo | ✅ Nativo |
| LinkedIn | ✅ Professional expected | ❌ Casual demais |

**Default para Meta cold audiences: UGC style.**
**Default para YouTube: polished com face-to-camera.**
