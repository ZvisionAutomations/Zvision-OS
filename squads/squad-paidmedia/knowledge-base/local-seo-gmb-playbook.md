| Item | Pontos | Verificar |

**5. Q&A (15 pontos)**
| Item | Pontos | Verificar |
|------|--------|-----------|
| Perguntas respondidas | 5 | Todas respondidas |
| Q&A seeding | 5 | Perguntas frequentes pre-populadas |
| Tempo de resposta | 5 | <48h |

## Local Market Competitive Analysis

### Framework de Analise
1. **Identificar top 5-10 concorrentes** por proximidade e relevancia
2. **Comparar metricas GMB:**
   - Rating medio
   - Numero de reviews
   - Response rate
   - Frequencia de posts
   - Completude do perfil
3. **Gap analysis:** onde somos melhores? Onde somos piores?
4. **SWOT local:** Strengths, Weaknesses, Opportunities, Threats

## Review Management Strategy

### Solicitar Reviews
1. **Pos-compra (D0-D1):** email ou SMS com link direto para review
2. **Follow-up (D7):** lembrete para quem nao avaliou
3. **In-person:** QR code no estabelecimento
4. **Meta:** 5-10 reviews novos/mes (consistencia > volume)

### Responder Reviews
**Positiva (4-5 estrelas):**
```
"Obrigado [Nome]! Ficamos felizes que [referencia ao que elogiou].
[Convite para voltar ou experimentar algo novo]. Ate a proxima!"
```

**Negativa (1-3 estrelas):**
```
"[Nome], lamentamos pela experiencia. [Reconhecer o problema especifico].
[Acao corretiva tomada]. Por favor entre em contato em [telefone/email]
para que possamos resolver diretamente. Obrigado pelo feedback."
```

### Regras
- Responder 100% das reviews negativas em <24h
- Responder 80%+ das reviews positivas em <48h
- Nunca ser defensivo ou discutir publicamente
- Levar conversas problematicas para privado

## Local Keyword Strategy

### Patterns de Keywords Locais
```
[servico] + [cidade]            → "dentista sao paulo"
[servico] + [bairro]            → "dentista pinheiros"
[servico] + perto de mim        → "dentista perto de mim"
[servico] + [qualificador]      → "melhor dentista sp"
[servico] + aberto agora        → "dentista aberto agora"
[servico] + [especializacao]    → "dentista implante sao paulo"
```

### Prioridade para Paid
1. High-intent local: "[servico] + [cidade]" com CPC viavel
2. "Perto de mim": capturar intencao imediata
3. Qualificadores: "melhor", "barato", "aberto agora"
4. Especializacao: nicho com menos competicao
