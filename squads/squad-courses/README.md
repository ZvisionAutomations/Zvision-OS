# Squad Courses

> Squad de criacao de cursos e material educacional. Cobre o espectro completo: cursos gravados em video, cursos escritos, apresentacoes didaticas, material de apoio, assessments, curriculum design e launch strategy. Da ideia ao curso publicado.

## Agentes

| Agente | Papel |
|--------|-------|
| Syllabus (courses-orqx) | Orquestrador — gerencia o ciclo completo de criacao de cursos |
| Compass (curriculum-designer) | Arquiteto de curriculo — learning paths, outcomes, pedagogia |
| Blueprint (lesson-architect) | Designer de licoes — estrutura modulos, secoes e progressao |
| Scribe (content-writer) | Escritor educacional — textos, ebooks, workbooks, handouts |
| Deck (slide-designer) | Designer de apresentacoes — slides para mentorias e workshops |
| Gauge (assessment-creator) | Designer de avaliacoes — quizzes, exercicios, certificacoes |
| Lens (production-director) | Diretor de producao — scripts, shot lists, recording guides |
| Launchpad (launch-strategist) | Estrategista de lancamento — plataforma, pricing, funnel |
