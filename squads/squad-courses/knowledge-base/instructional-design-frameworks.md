# Instructional Design Frameworks

Reference guide for selecting and applying the right framework for each course context.

---

## Framework Comparison Table

| Framework | Best For | Key Steps | Output |
|-----------|----------|-----------|--------|
| **ADDIE** | Full course builds, corporate training | Analyze → Design → Develop → Implement → Evaluate | Complete course with evaluation loop |
| **Bloom's Taxonomy** | Writing objectives, assessments | Remember → Understand → Apply → Analyze → Evaluate → Create | Verb-mapped learning objectives |
| **Gagné's 9 Events** | Single lesson/module design | 9 instructional events sequence | Structured lesson with measurable outcome |
| **Backward Design** | Outcome-first curriculum | Identify outcomes → Assessments → Learning activities | Aligned curriculum map |
| **Microlearning** | Busy professionals, mobile learners | Chunk → Focus → Reinforce | 3–10 min lesson units |
| **Spaced Repetition** | Retention-heavy content | Schedule review intervals | Spaced review system |

---

## ADDIE — Step-by-Step

| Phase | Questions to Answer | Key Deliverable |
|-------|---------------------|-----------------|
| **Analyze** | Who is the learner? What problem are we solving? What gaps exist? | Learner profile + needs analysis |
| **Design** | What are the outcomes? What sequence? What assessments? | Curriculum map + assessment plan |
| **Develop** | What content, materials, activities? | Lessons, workbooks, videos |
| **Implement** | How is it delivered? What platform? | Live or async course |
| **Evaluate** | Did learners achieve outcomes? What improves? | Feedback report + iteration plan |

## Bloom's Taxonomy — 6 Levels

| Level | Cognitive Skill | Action Verbs | Assessment Type |
|-------|----------------|--------------|-----------------|
| 1 — Remember | Recall facts | List, define, identify | Multiple choice, fill-in |
| 2 — Understand | Explain in own words | Summarize, describe, explain | Short answer, paraphrase |
| 3 — Apply | Use in new situations | Solve, use, execute, implement | Case study, exercise |
| 4 — Analyze | Break into parts | Compare, differentiate, examine | Breakdown exercise |
| 5 — Evaluate | Make judgements | Critique, justify, assess | Rubric-based review |
| 6 — Create | Produce new work | Design, build, compose | Project, portfolio |

**Rule:** Write all learning objectives starting with a Bloom's action verb. Match assessment type to the level.

## Gagné's 9 Events of Instruction

| Event | Purpose | Example |
|-------|---------|---------|
| 1. Gain attention | Activate focus | Surprising statistic, story, question |
| 2. State objectives | Set expectations | "By the end, you'll be able to..." |
| 3. Stimulate recall | Connect prior knowledge | "You already know X. Now we'll add Y." |
| 4. Present content | Teach the concept | Video, text, demo |
| 5. Provide guidance | Scaffold learning | Worked example, template |
| 6. Elicit performance | Practice | Exercise, quiz |
| 7. Provide feedback | Correct + reinforce | Answer key, comments |
| 8. Assess performance | Measure outcome | Test, project submission |
| 9. Enhance retention | Transfer to real life | Action plan, journaling prompt |

## Backward Design (Wiggins & McTighe)

1. **Identify desired results** — What will students know/do/be?
2. **Determine acceptable evidence** — How will you know they got it? (assessment first)
3. **Plan learning experiences** — What activities lead to that evidence?

**When to use:** Whenever there's a risk of teaching content that doesn't connect to real outcomes.

---

## Microlearning

- **Unit size:** 3–10 minutes per lesson
- **One concept per unit** — no cramming
- **Mobile-first format** — works on any device
- **Reinforcement:** Pair with spaced repetition
- **Best for:** Skill-based, compliance, product training

---

## Spaced Repetition

| Interval | Review Timing |
|----------|--------------|
| First review | 1 day after learning |
| Second review | 3 days later |
| Third review | 7 days later |
| Fourth review | 21 days later |
| Long-term | Monthly |

**Implementation:** Embed recap quizzes at module start. Use "Previously..." recaps. Assign journaling prompts.
