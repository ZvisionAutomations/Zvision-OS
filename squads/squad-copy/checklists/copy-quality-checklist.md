# Copy Quality Checklist

## Uso
Aplicar em TODA peca de copy antes da publicacao. Cobre estrategia, persuasao, voice, clareza e tecnico.

## Estrategia (Pass 1)
- [ ] Objetivo claro e unico
- [ ] Audiencia correta
- [ ] Awareness level adequado
- [ ] Message hierarchy respeitada
- [ ] CTA alinhado com objetivo
- [ ] Brief compliance verificada

## Persuasao (Pass 2)
- [ ] Headline captura em 3 segundos
- [ ] Emocao antes de logica
- [ ] Benefits (nao features)
- [ ] Provas em 3+ camadas (social, authority, demo)
- [ ] Objecoes top 3 endereçadas
- [ ] Future pacing presente
- [ ] Min 4 de 7 triggers de Cialdini
- [ ] Urgencia/escassez real (se usada)

## Voice (Pass 3)
- [ ] Voice attributes presentes
- [ ] Zero never-words
- [ ] Tom adequado para situacao
- [ ] Consistencia ao longo da peca
- [ ] Expressoes assinatura (se cabem)

## Clareza e Impacto (Pass 4)
- [ ] Zero fluff words
- [ ] Voz ativa predominante
- [ ] Concreto > abstrato
- [ ] Frases < 20 palavras (media)
- [ ] Paragrafos < 4 linhas
- [ ] Ritmo variado
- [ ] Power phrases em cada secao
- [ ] Leitura em voz alta aprovada
- [ ] Scanning test: subheadlines contam a historia

## Tecnico (Pass 5)
- [ ] Zero erros ortograficos/gramaticais
- [ ] Links funcionam
- [ ] Character limits respeitados
- [ ] Formatacao correta
- [ ] Compliance/legal ok
- [ ] Mobile-friendly

## Etica
- [ ] Todos os claims verificaveis
- [ ] Social proof real
- [ ] Escassez real
- [ ] Teste do jornal: passaria?
- [ ] Zero dark patterns

## Score Minimo para Publicacao
- Estrategia: 100% (todos obrigatorios)
- Persuasao: >= 80% dos itens
- Voice: >= 90% dos itens
- Clareza: >= 80% dos itens
- Tecnico: 100% (todos obrigatorios)
- Etica: 100% (todos obrigatorios)
