# Conversion Copy Checklist

## Uso
Aplicar em TODA peca de copy com objetivo de conversao (landing pages, emails, ads, opt-in pages, sales pages). Cobre desde estrategia ate otimizacao pos-publicacao.

## Pre-Producao
- [ ] Objetivo de conversao definido (1 unico objetivo)
- [ ] Metrica principal identificada (CR, CTR, CPL, ROAS)
- [ ] Audiencia segmentada com awareness level
- [ ] Pesquisa de audiencia consultada (dores, desejos, objecoes)
- [ ] Ofertas/CTAs definidos antes de escrever
- [ ] Competitors analisados (o que funciona/nao funciona)

## Headline & Hook
- [ ] Headline captura atencao em < 3 segundos
- [ ] Headline especifica (numeros, resultados, prazos)
- [ ] Headline fala de BENEFICIO (nao feature)
- [ ] Subheadline complementa (nao repete)
- [ ] Hook adequado ao awareness level
- [ ] Min 3 variantes de headline criadas para teste

## Corpo da Copy
- [ ] Problema articulado antes da solucao
- [ ] Benefits > Features (ratio minimo 3:1)
- [ ] Cada beneficio tem prova de suporte
- [ ] Transicoes fluidas (Slippery Slide)
- [ ] Bullet points para scanning rapido
- [ ] Paragrafos curtos (max 4 linhas)
- [ ] Sem jargao desnecessario

## Persuasao
- [ ] Min 4 principios de Cialdini aplicados
- [ ] Social proof presente e especifico
- [ ] Risk reversal claro (garantia, trial, etc.)
- [ ] Objecoes top 3 endereçadas
- [ ] Loss framing equilibrado (max 30% do total)
- [ ] Urgencia/escassez REAL (se usada)
- [ ] Future pacing presente

## CTA (Call to Action)
- [ ] CTA visivel sem scroll (above the fold)
- [ ] CTA repetido em pontos estrategicos
- [ ] CTA usa formula: Acao + Beneficio
- [ ] Micro-copy de suporte presente (friction reducers)
- [ ] CTA contrasta visualmente com o entorno
- [ ] Alternativa de baixo compromisso disponivel
- [ ] Zero ambiguidade no que acontece ao clicar

## Social Proof
- [ ] Testimonials especificos (numeros, nomes, resultados)
- [ ] Social proof de SEMELHANTES a audiencia
- [ ] Min 3 camadas de prova (testimonial + dados + autoridade)
- [ ] Logos/badges de confianca presentes (se aplicavel)
- [ ] Social proof posicionado ANTES do CTA principal
- [ ] Reviews/ratings incluidos (se existem)

## Pricing (se aplicavel)
- [ ] Anchoring com valor real de referencia
- [ ] Value stack antes do preco
- [ ] Framing por periodo menor (por dia/mes vs total)
- [ ] Comparacao contextualizada (custo de um cafe)
- [ ] Planos com opção decoy (se 3+ planos)
- [ ] ROI calculado e apresentado
- [ ] Garantia proxima ao preco

## Email Especifico
- [ ] Subject line < 50 caracteres
- [ ] Preview text otimizado (nao repete subject)
- [ ] From name reconhecivel
- [ ] Primeiro paragrafo gera curiosidade
- [ ] Um unico CTA por email (regra 80% dos casos)
- [ ] PS line presente (segundo CTA)
- [ ] Unsubscribe visivel e facil
- [ ] Teste em mobile antes de enviar

## Ads Especifico
- [ ] Character limits respeitados por plataforma
- [ ] Hook nos primeiros 3 segundos (video) ou 5 palavras (texto)
- [ ] Congruencia ad → landing page (mensagem consistente)
- [ ] CTA claro dentro do limite de caracteres
- [ ] Sem promessas que LP nao cumpre
- [ ] Variantes para teste A/B (min 3)
- [ ] UTMs configurados para tracking

## Landing Page Especifico
- [ ] Zero navegacao externa (sem menu, sem links de saida)
- [ ] Page load < 3 segundos
- [ ] Hero section conta a historia completa
- [ ] Scanning test: subheadlines contam a historia sozinhas
- [ ] Formulario com minimo de campos necessarios
- [ ] Thank you page otimizada (proximo passo claro)
- [ ] Mobile responsive testado

## Tecnico
- [ ] Links funcionam (todos testados)
- [ ] Tracking pixels instalados
- [ ] Formularios enviam corretamente
- [ ] Redirects configurados
- [ ] Pagina de obrigado funciona
- [ ] Integracao com CRM/email tool ok
- [ ] Cross-browser testado

## Etica
- [ ] Claims verificaveis
- [ ] Escassez real
- [ ] Social proof autentico
- [ ] Sem dark patterns
- [ ] Compliance regulatorio (CONAR/CDC/FTC)
- [ ] Teste do jornal: passaria?

## Score Minimo para Publicacao
- Pre-Producao: 100% (todos obrigatorios)
- Headline & Hook: >= 85%
- Corpo: >= 80%
- Persuasao: >= 80%
- CTA: 100% (todos obrigatorios)
- Social Proof: >= 75%
- Secao Especifica (Email/Ads/LP): >= 90%
- Tecnico: 100% (todos obrigatorios)
- Etica: 100% (todos obrigatorios)
