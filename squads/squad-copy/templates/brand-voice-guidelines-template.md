## 6. Regras de Escrita

### Gramatica e Estilo

| Regra | Padrao |
|-------|--------|
| Tratamento | [Voce / Tu / Senhor(a)] |
| Plural inclusivo | [Nos / A gente / Nossa equipe] |
| Contrações | [Sempre / As vezes / Nunca] |
| Virgula de Oxford | [Sim / Nao] |
| Ponto final em titulos | [Sim / Nao] |
| Numeros | [Extenso ate 10 / Sempre numeral / Extenso ate 100] |
| Porcentagens | [% / por cento] |
| Datas | [DD/MM/AAAA / dia de mes de ano] |
| Horas | [24h / 12h AM/PM] |
| Moeda | [R$ / BRL / Reais] |

### Formatacao

| Elemento | Regra |
|----------|-------|
| Headlines | [Title Case / Sentence case / UPPERCASE] |
| Subheadlines | [Title Case / Sentence case] |
| Bullets | [Ponto final / Sem ponto / Depende do tamanho] |
| Bold | [Para enfase / Sparingly / Liberalmente] |
| Italico | [Para citacoes / Para enfase / Evitar] |
| Emojis | [Liberados / Moderados / Proibidos] |
| Hashtags | [Camelcase / lowercase / UPPERCASE] |
