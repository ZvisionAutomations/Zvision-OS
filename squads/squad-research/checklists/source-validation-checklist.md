# Source Validation Checklist

Aplicar para CADA fonte utilizada em pesquisa ANALYZE ou superior.

## Classificacao
- [ ] Source tier classificado (Tier 1-5)
- [ ] Tipo de fonte identificado (primaria vs secundaria)

## Credibilidade do Autor
- [ ] Autor/organizacao identificado e verificavel
- [ ] Expertise do autor no dominio confirmada
- [ ] Track record do autor/org avaliado
- [ ] Afiliacao institucional verificada

## Qualidade dos Dados
- [ ] Data de publicacao dentro da janela aceitavel
- [ ] Metodologia descrita e documentada
- [ ] Sample size adequado (para dados empiricos)
- [ ] Dados sao verificaveis ou reproduziveis

## Bias Check
- [ ] Conflitos de interesse identificados (financeiro, politico, ideologico)
- [ ] Fonte patrocinada? Se sim, por quem?
- [ ] Linguagem emocional/persuasiva identificada
- [ ] Dados seletivos? (cherry picking detectado?)

## Consistencia
- [ ] Dados consistentes com outras fontes independentes
- [ ] Se inconsistente: divergencia documentada e investigada
- [ ] Nao ha circular sourcing (fonte A cita B que cita A)

## Documentacao
- [ ] Fonte completamente referenciada (titulo, autor, data, URL)
- [ ] Tier e credibilidade documentados
- [ ] Biases potenciais anotados
- [ ] Contexto de uso registrado (que claim esta fonte suporta)
