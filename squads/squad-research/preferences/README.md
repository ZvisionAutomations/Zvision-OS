## Delivery Preferences

### Formato Padrao por Audiencia
- **C-Level / Board:** Executive Briefing (2-3 paginas max)
- **Strategy Team:** Research Report completo
- **Product Team:** Insights focados + recomendacoes actionable
- **Sales Team:** Battle cards + competitive dossier
- **Marketing Team:** Audience personas + trend reports

### Idioma
- Reports internos: Portugues (BR)
- Reports para investidores: English
- Termos tecnicos: Sempre em English

## Quality Standards

### Confidence Levels
- **HIGH (>80%):** 3+ sources Tier 1-2, dados recentes (<6 meses), triangulacao completa
- **MEDIUM (50-80%):** 2+ sources Tier 2-3, dados <12 meses, parcialmente triangulado
- **LOW (<50%):** Sources limitadas ou Tier 4-5, dados >12 meses, hipotese nao validada

### SLA por Depth
- SCAN: 30 minutos
- ANALYZE: 4 horas
- DEEP DIVE: 2 dias uteis
- DEFINITIVE: 5 dias uteis

## Agent Activation Order (Sugerido)

Para pesquisa completa, ativar nesta ordem:
1. **Prism** — Triage e planejamento
2. **Sage** — Pesquisa profunda (base de conhecimento)
3. **Pulse** + **Hawk** + **Scope** — Em paralelo (audiencia + competitivo + mercado)
4. **Horizon** — Tendencias e cenarios (precisa dos dados anteriores)
5. **Loom** — Sintese final (precisa de todos os inputs)
6. **Prism** — Review e entrega
