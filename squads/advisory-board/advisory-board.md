# Advisory Board

Conselho virtual de 11 pensadores estrategicos e lideres de classe mundial.

## Inicio Rapido

```
@board-chair        # Ativar o orquestrador
*diagnose           # Triagem da sua questao estrategica
*board-meeting      # Sessao completa do conselho
*decision-framework # Analise estruturada de decisao
```
