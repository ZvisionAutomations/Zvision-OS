# Story Structure Reference

> Version: 1.0.0 | Quick-reference guide for narrative structure patterns across all frameworks

## Universal Story DNA

Every story, regardless of framework, contains these elements:

| Element | Description | Framework Term |
|---------|-------------|---------------|
| **Status Quo** | The world before change | Ordinary World (Campbell), You (Harmon), Opening Image (Snyder), What Is (Duarte), Normal (Hall), And (Howell) |
| **Disruption** | Something breaks the status quo | Call to Adventure (Campbell), Need (Harmon), Catalyst (Snyder), Explosion (Hall), But (Howell) |
| **Journey** | The struggle through the unknown | Road of Trials (Campbell), Search (Harmon), Fun & Games (Snyder), Toggle (Duarte) |
| **Crisis** | The darkest moment / death of old self | Atonement (Campbell), Find/Take (Harmon), All Is Lost (Snyder), Story of Now (Ganz) |
| **Transformation** | The old self dies, new self emerges | Apotheosis (Campbell), Change (Harmon), Break into Three (Snyder), New Normal (Hall), Therefore (Howell) |
| **Return** | Back to the world, but changed | Return with Elixir (Campbell), Return (Harmon), Final Image (Snyder), New Bliss (Duarte), Encore (Howell) |

## Three-Act Structure Mapping

| Act | Campbell | Harmon | Snyder | Duarte | Hall | Ganz |
|-----|---------|--------|--------|--------|------|------|
| **Act 1: Setup** | Departure (stages 1-5) | You, Need, Go | Opening Image → Break into Two | What Is (opening) | Normal | Story of Self |
| **Act 2: Confrontation** | Initiation (stages 6-11) | Search, Find, Take | B Story → Dark Night | What Is / What Could Be toggle | Explosion | Story of Us |
| **Act 3: Resolution** | Return (stages 12-17) | Return, Change | Break into Three → Final Image | Call to Action + New Bliss | New Normal | Story of Now |

## Emotional Arc Patterns

### The Classic Arc (most frameworks)
```
Hope → Fear → Struggle → Despair → Breakthrough → Triumph
```

### The Sparkline (Duarte)
```
Pain → Hope → Pain → Hope → Pain → Hope (escalating) → S.T.A.R. → New Bliss
```

### The Pitch Arc (Klaff)
```
Intrigue → Desire → Tension → Scarcity → Urgency → Commitment
```

### The Movement Arc (Ganz)
```
Personal courage → Collective solidarity → Shared urgency → Hope → Action
```

## Opening Techniques (First 30 Seconds)

| Technique | Framework | Example |
|-----------|-----------|---------|
| In medias res (start in the middle) | Universal | Open with the crisis, then flashback |
| The question | Campbell | "Have you heard the call?" |
| The declaration | Ganz | "We will not be silent" |
| The ABT | Howell | "We want X, AND we have Y, BUT Z, THEREFORE..." |
| The challenge | Klaff | "Most people get this completely wrong" |
| The contrast | Duarte | "Here's the world as it is... here's what it could be" |
| The specific moment | Hall | "It was 3:47 AM when the phone rang..." |
| The invitation to play | Johnstone | "Stop trying to be good. Let's play." |

---

## Closing Techniques (Last 30 Seconds)

| Technique | Framework | Purpose |
|-----------|-----------|---------|
| Return to Opening Image | Snyder | Show transformation through visual contrast |
| The New Bliss | Duarte | Paint the vivid future after action is taken |
| The Rallying Cry | Ganz | One sentence they will repeat and share |
| The Call to Action | Universal | Specific, achievable, immediate next step |
| The Boon Shared | Campbell | Hero returns with gift for the community |
| Story On! | Howell | Energy and momentum for the next chapter |
| The Echo | Harmon | Mirror Step 1, but different — proof of change |

## Conflict Types Across Frameworks

| Type | Description | Best Framework |
|------|-------------|---------------|
| Person vs. Self | Internal struggle, identity crisis | Campbell (Atonement), Hall (Self-Stories) |
| Person vs. Person | Direct antagonist, competition | Snyder (Bad Guys Close In), Klaff (Frame Control) |
| Person vs. Society | System, institution, status quo | Ganz (Public Narrative), Campbell (Threshold Guardians) |
| Person vs. Nature | Environment, fate, circumstance | Campbell (Road of Trials), Snyder (Dude with a Problem) |
| Person vs. Technology | Modern challenges, disruption | Duarte (What Is / What Could Be) |
| Person vs. Inertia | Resistance to change, comfort zone | Howell (The "But"), Johnstone (Blocks) |

## Story Length Guidelines

| Format | Duration | Recommended Framework |
|--------|----------|----------------------|
| Elevator pitch | 30-60 seconds | ABT (Howell) |
| Business story | 60-90 seconds | Normal-Explosion-New Normal (Hall) |
| Pitch meeting | 10-20 minutes | STRONG Method (Klaff) |
| Presentation | 20-45 minutes | Sparkline (Duarte) |
| TV episode | 22-60 minutes | Story Circle (Harmon) |
| Screenplay | 90-120 minutes | Beat Sheet (Snyder) |
| Movement speech | 5-15 minutes | Public Narrative (Ganz) |
| Epic narrative | Unlimited | Hero's Journey (Campbell) |

## Common Story Failures and Fixes

| Failure | Symptom | Fix | Agent |
|---------|---------|-----|-------|
| No conflict | Story feels flat, boring | Find the "BUT" — add tension | park-howell |
| No transformation | Characters unchanged | Ensure death/rebirth at midpoint | dan-harmon |
| Hero is the brand | Audience doesn't connect | Make customer the hero, brand the guide | park-howell |
| Data without narrative | Audience disengages | Apply DataStory framework | nancy-duarte |
| Generic stories | Nothing sticks | Add specificity — concrete details | kindra-hall |
| Creative block | Can't start or continue | Remove blocks with improv exercises | keith-johnstone |
| Abstract values | No emotional response | Ground in specific moments of choice | marshall-ganz |
| Neocortex targeting | Pitch fails, buyer goes cold | Activate croc brain first | oren-klaff |
| Missing beats | Story feels incomplete | Map to beat sheet and fill gaps | blake-snyder |
| No mythic resonance | Story feels disposable | Map archetypes and universal patterns | joseph-campbell |

## Framework Compatibility Matrix

Frameworks that work well together:

| Combination | Use Case |
|-------------|----------|
| ABT + 4 Stories | Brand storytelling suite |
| Hero's Journey + Beat Sheet | Feature film development |
| Story Circle + Beat Sheet | TV series with film-quality episodes |
| Sparkline + DataStory | Data-driven keynote presentations |
| Public Narrative + Sparkline | Movement communications + visual delivery |
| STRONG + Sparkline | Pitch with presentation support |
| Status + Story Circle | Scene-level TV writing |
| ABT + Public Narrative | Organization change communication |
| 4 Stories + Story Cycle | Complete business narrative system |

Frameworks that conflict (use one OR the other):

| Conflict | Reason |
|----------|--------|
| STRONG (Klaff) + Sparkline (Duarte) for same pitch | Frame dominance vs audience empowerment — different philosophies |
| Beat Sheet (Snyder) + Impro (Johnstone) | Structure vs spontaneity — choose approach first |
