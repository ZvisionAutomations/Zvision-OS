# /tldr — Salvar resumo da sessão

Base directory for this skill: o diretorio atual do vault

## O que fazer

1. Resumir a conversa atual em:
   - **Decisões tomadas** — o que foi decidido e por quê
   - **Coisas para lembrar** — contexto importante que pode ser útil depois
   - **Próximas ações** — o que ficou pendente

2. Determinar a pasta mais relevante para salvar:
   - Se sobre um projeto → `projetos/`
   - Se sobre um cliente → `clientes/`
   - Se sobre um sistema/automação → `sistemas/`
   - Se sobre pesquisa/ferramenta → `pesquisa/`
   - Se decisão importante → `decisoes/`
   - Se pessoal → `pessoal/`
   - Se não couber em nenhuma → `inbox/`

3. Salvar como `[pasta]/[titulo-descritivo].md` ou adicionar à nota existente se o assunto já tiver uma

4. Atualizar a memória do Claude se houver informação relevante para futuras sessões
