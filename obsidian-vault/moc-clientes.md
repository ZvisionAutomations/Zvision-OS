# Clientes

A Zvision opera com modelo de managed service contínuo, não entrega pontual.
Cada cliente é uma relação de longo prazo — o [[sop-onboarding-cliente]] define
como essa relação começa, e o [[checklist-qualidade-preentrega]] garante que
cada entrega mantém o padrão.
