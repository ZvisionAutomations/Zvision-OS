# Claude Code Mastery Squad

O squad de Claude Code domina todos os aspectos do ecossistema:
o [[config-engineer]] para configuração e hooks, o [[mcp-integrator]]
para novos servidores MCP e o [[skill-craftsman]] para criação de
skills. O [[zvision-os]] em si é o maior projeto deste squad.

## Agentes principais
- [[claude-orqx]] (Orion) — orquestrador
- [[config-engineer]] — configuração, hooks e settings
- [[mcp-integrator]] — integração de MCP servers
- [[skill-craftsman]] — criação e manutenção de skills
- [[agent-forger]] — criação de novos agentes

## Domínios
- [[moc-claude-skills]] — biblioteca de skills criadas
- [[moc-claude-mcps]] — MCPs instalados e configurados
- [[moc-claude-workflows]] — workflows automatizados

## Quando usar este squad
Criar novos agentes ou skills, configurar MCPs, automatizar
workflow do Claude Code, debugar hooks. Toda mudança em
[[zvision-os]] que afete a camada de IA passa por aqui.
Ver [[unified-intelligence-stack]] para o estado atual do sistema.
