# Dev Squad

O squad de desenvolvimento transforma specs do [[squad-design]] em código
world-class usando a stack nativa da Zvision: Vanilla HTML, CSS com
[[css-variables-system]] como linguagem de design, e JavaScript ES6+ modular.
O [[dev-orqx]] (Forge Prime) orquestra os 7 especialistas, garantindo que
nada vai ao ar sem passar pelo [[quality-gate]] e pelo [[security-dev]].

## Agentes principais
- [[dev-orqx]] (Forge Prime) — orquestrador, delega ao especialista certo
- [[architect]] (Blueprint) — decisões de arquitetura e tech stack
- [[developer]] (Coder) — implementação de HTML/CSS/JS e Node.js
- [[ux-engineer]] (Pixel) — UI pixel-perfect, componentes visuais
- [[security-dev]] (Shield) — auditoria OWASP, segurança do código

## Domínios
- [[moc-dev-stack]] — Vanilla JS, CSS Variables, Node.js, Vercel
- [[moc-dev-workflows]] — feature-build-cycle, bug-fix-cycle, deploy-cycle
- [[moc-dev-quality]] — checklists, code review, definition of done

## Quando usar este squad
Qualquer trabalho de código que sai em nome da Zvision — landing page,
feature nova, correção de bug, deploy — passa pelo [[dev-orqx]] antes de
ir ao ar. O [[squad-design]] fornece o spec visual; o [[security-dev]]
integra com o [[squad-cybersecurity]] para auditorias profundas quando necessário.
Ferramenta de invocação: `/dev:agents:dev-orqx`.
