# Courses Squad

O squad de cursos opera com [[curriculum-designer]] para estrutura
pedagógica, [[lesson-architect]] para design de aulas e
[[assessment-creator]] para avaliações. Todo curso criado segue
o [[moc-courses-methodology]] como padrão de qualidade.

## Agentes principais
- [[courses-orqx]] — orquestrador
- [[curriculum-designer]] — design curricular e estrutura pedagógica
- [[lesson-architect]] — arquitetura de aulas e módulos
- [[assessment-creator]] — avaliações e certificações
- [[launch-strategist]] — estratégia de lançamento educacional

## Domínios
- [[moc-courses-methodology]] — metodologia de criação de cursos
- [[moc-courses-zvision]] — treinamentos internos da Zvision
- [[moc-courses-clients]] — cursos criados para clientes

## Quando usar este squad
Criar treinamento para cliente, desenvolver curso da Zvision,
estruturar onboarding educacional, lançar produto de conhecimento.
O [[squad-storytelling]] fornece a narrativa; o courses squad
estrutura o aprendizado.
