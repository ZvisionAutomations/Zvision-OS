# Courses Methodology

## Princípio pedagógico
O [[curriculum-designer]] opera com o princípio de aprendizagem
ativa: cada módulo termina com aplicação prática, não apenas teoria.
O [[lesson-architect]] estrutura: conceito → exemplo real → exercício → feedback.

## Estrutura padrão de curso
Módulo 0: contexto e promessa clara de transformação.
Módulos 1-N: conteúdo em sequência de complexidade crescente.
Módulo final: projeto integrador que prova a transformação.
O [[assessment-creator]] valida que os exercícios testam aplicação,
não memorização.

## Aplicação atual
A Zvision usa este squad para onboarding de clientes — transformar
a entrega de automação em processo educacional que o cliente
consegue replicar e manter. Isso aumenta retenção e reduz suporte.

## Launch strategy
O [[launch-strategist]] define: cursos para o mercado externo
entram no roadmap após validação interna. Primeiro, usa-se o
curso como diferencial de entrega para clientes; depois, como
produto independente via [[squad-commercial]].
