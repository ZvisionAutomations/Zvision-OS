# Commercial Squad

O squad comercial opera com [[cs-sales-closer]] para fechamento,
[[cs-lead-generation-strategist]] para prospecção e [[cs-funnel-architect]]
para estruturar o pipeline. O estado atual do pipeline vive em
[[pipeline-abril-2026]] — 94 leads prontos para outbound.

## Agentes principais
- [[commercial-orqx]] (Pipeline) — orquestrador
- [[cs-sales-closer]] — técnicas de fechamento
- [[cs-lead-generation-strategist]] — geração e qualificação de leads
- [[cs-funnel-architect]] — arquitetura de funil
- [[cs-crm-specialist]] — gestão do CRM e pipeline

## Domínios
- [[moc-commercial-pipeline]] — estado do pipeline e estratégia
- [[moc-commercial-outbound]] — metodologia de outbound
- [[moc-commercial-deals]] — histórico de deals fechados

## Quando usar este squad
Estruturar oferta para cliente, definir precificação, criar proposta
comercial, qualificar leads, gerenciar pipeline. O [[squad-hormozi]]
estrutura a oferta; o commercial squad executa a venda.
