# Product Roadmap

## Produtos ativos
O [[zvision-landing]] está em produção no Vercel — hardened,
com todos os headers de segurança do [[security-audit-landing-2026-03]].
O [[zvision-crm]] está em produção com Supabase, auditado pelo [[crm-audit-2026-04]].
O [[zvision-os]] é este sistema operacional — base de tudo.

## Prioridade atual
O [[ps-product-strategist]] define: primeiro cliente fechado
antes de qualquer nova feature. Nenhum novo produto é iniciado
sem [[obsidian-vault/specs/]] aprovada e story no pipeline do [[squad-dev]].

## Visão de produto (6 meses)
Landing page builder packaged para clínicas (produto, não serviço).
Automação de agendamento como SaaS B2B.
O [[squad-finance]] valida unit economics antes do [[squad-dev]] implementar.

## Processo de discovery
Toda nova feature começa com [[ps-discovery-lead]] validando
com usuários reais antes do [[ps-product-analyst]] escrever a spec.
O [[squad-dev]] só implementa depois da spec aprovada pelo [[squad-product]].
