# Hormozi Squad

O squad Hormozi opera com o framework $100M Offers como base:
toda oferta da Zvision é estruturada para maximizar o valor percebido
e minimizar o risco do cliente. O [[squad-commercial]] executa a venda;
este squad garante que o que é vendido é irresistível.

## Framework central
A equação de valor do Hormozi: (Dream Outcome × Perceived Likelihood
of Achievement) / (Time Delay × Effort and Sacrifice). O [[squad-copy]]
traduz essa equação em linguagem que o [[publico-alvo-zvision]] entende.

## Domínios
- [[moc-hormozi-offers]] — ofertas estruturadas por serviço
- [[moc-hormozi-pricing]] — modelos de pricing testados
- [[moc-hormozi-scaling]] — estratégias de escala e leverage

## Ofertas ativas
A [[oferta-landing-pages-v1]] estruturada para clínicas e imobiliárias
usa o tier entry R$1.200 como pé-na-porta para o core R$2.400,
que naturalmente abre para automação via [[squad-commercial]].

## Quando usar este squad
Estruturar qualquer oferta antes de apresentar ao cliente,
revisar precificação, criar upsell e cross-sell. Toda oferta
passa aqui antes de chegar ao [[squad-copy]] para ser escrita.
