# Design Squad

O squad de design opera com [[atomic-design]] como metodologia base,
[[brad-frost]] como arquiteto de design system e [[dave-malouf]]
para DesignOps. Toda decisão visual é documentada em [[moc-design-tokens]]
antes de entrar no código.

## Agentes principais
- [[design-orqx]] (Nexus) — orquestrador
- [[dx-ui-designer]] — design de componentes e layouts
- [[dx-ux-strategist]] — estratégia e pesquisa com usuário
- [[dx-design-system-architect]] — design system e tokens
- [[dx-accessibility-specialist]] — acessibilidade e WCAG

## Domínios
- [[moc-design-system]] — componentes, tokens e padrões
- [[moc-design-ux]] — estratégia de experiência do usuário
- [[moc-design-clients]] — projetos de design para clientes

## Quando usar este squad
Auditar ou criar interfaces, construir design system, revisar
qualidade visual — tudo passa por aqui. O [[squad-dev]] recebe
o design aprovado pelo [[dx-ui-designer]] antes de implementar.
Ferramentas: /design-review, /design-shotgun, /design-html.
