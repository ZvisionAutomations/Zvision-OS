# Research Squad

O squad de pesquisa e dados opera com [[deep-researcher]] para
análise de mercado, [[competitive-intelligence]] para benchmarks
e [[data-synthesizer]] para transformar dados brutos em decisões.
Todo insight é documentado no [[moc-research-market]] antes de
alimentar outros squads.

## Agentes principais
- [[research-orqx]] (Prism) — orquestrador
- [[deep-researcher]] — pesquisa profunda e desk research
- [[competitive-intelligence]] — análise de competidores
- [[market-analyst]] — tendências e oportunidades de mercado
- [[data-synthesizer]] — síntese de dados em insights

## Domínios
- [[moc-research-market]] — mercado de automação e IA no Brasil
- [[moc-research-clients]] — pesquisa por segmento de cliente
- [[moc-research-competitors]] — mapeamento competitivo

## Quando usar este squad
Pesquisa de mercado para cliente, análise competitiva, definição
de métricas de sucesso. O [[squad-commercial]] usa os insights
aqui para calibrar propostas; o [[squad-product]] usa para roadmap.
