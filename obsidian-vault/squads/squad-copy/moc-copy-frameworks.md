# Copy Frameworks

## Before-After-Bridge (BAB)
O framework mais usado pelo [[squad-copy]] para abertura de landing pages:
pinta o estado atual (dor), mostra o estado futuro (transformação)
e posiciona a Zvision como a ponte. O [[copy-chief]] valida
que o "before" ressoa com o [[publico-alvo-zvision]] antes de publicar.

## Problem-Agitate-Solution (PAS)
Usado pelo [[conversion-writer]] para emails e scripts de outbound.
O agite é o diferencial — sem torcer a faca na ferida, o copy
é esquecível. Ver aplicação em [[outbound-clinicas]].

## Framework Hormozi aplicado ao copy
O [[squad-hormozi]] estrutura a oferta; o [[squad-copy]] traduz para
linguagem de transformação. Regra: nunca falar do mecanismo
("automação de processos"), sempre falar do resultado
("sua operação roda sem você estar presente").

## Tom proibido na Zvision
O [[copy-chief]] bloqueia qualquer texto com: "solução", "parceria",
"agregar valor", "nicho", "inovar", "revolucionar".
Alternativas: resultado concreto, número específico, prazo real.
