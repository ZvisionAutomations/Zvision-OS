# Plugin: Tour Virtual & Apresentação IA

## O que faz
Quando um imóvel é cadastrado, gera automaticamente roteiro de tour e material de apresentação.

## Fluxo
`
Imóvel cadastrado no sistema
  → IA analisa fotos + características
  → Gera: descrição SEO, roteiro de tour, pontos de destaque
  → Cria apresentação visual (slides/PDF)
  → Publica nos portais com copy otimizado
`

## MCP
- **Playwright** (mcp__playwright__*) — acesso e publicação automática nos portais
- **EXA** — pesquisa de comparativos de mercado para ancoragem de preço

## Agentes Zvision
- @vivid (imagem mental) — escreve descrição cinematográfica
- @anchor (ancoragem) — posiciona preço vs mercado

## Output
- Descrição otimizada para portal (SEO)
- Roteiro de 5-7 minutos de tour para vídeo
- PDF de apresentação para WhatsApp
