# Plugin: Captação de Leads Multicanal

## O que faz
Unifica leads de todos os canais em um pipeline único com qualificação automática.

## Canais Integrados
- WhatsApp Business
- Instagram DM (via n8n)
- Google Ads (Lead Form)
- Meta Ads (Lead Ads)
- Portal imobiliário (webhook)
- Site próprio (formulário)

## Fluxo de Qualificação
`
Lead capturado em qualquer canal
  → Normalização de dados (nome, telefone, interesse)
  → Perguntas de qualificação via IA (3-5 perguntas)
  → Score calculado
  → Distribuição para corretor responsável
  → CRM atualizado automaticamente
`

## Integrações
- n8n (orquestração)
- Meta Ads API
- Google Ads API
- WhatsApp Cloud API

## MCP
Nenhum necessário — operação via n8n + APIs nativas
