# Plugin: Portal do Cliente Automatizado

## O que faz
Clientes acompanham processos, recebem updates e enviam documentos via WhatsApp.

## Funcionalidades para o Cliente
- Consulta de andamento processual em linguagem simples (não jurídica)
- Recebimento de notificações de movimentações importantes
- Envio de documentos solicitados diretamente pelo WhatsApp
- Consulta de agenda de audiências
- Comunicação direta com o advogado (triada pela IA)

## Tradução Jurídica → Linguagem Simples
`
Movimentação: "Juntada de contestação pela parte ré"
→ Tradução IA: "O outro lado respondeu ao processo. Seu advogado está analisando 
  a resposta deles. Em breve você receberá um resumo."
`

## Integrações
- WhatsApp Cloud API
- n8n (automação de mensagens)
- Base processual do escritório

## Agentes Zvision
- @rhythm (daily emails) — cadência de comunicação com cliente
- @stream (fluidez) — linguagem clara e sem juridiquês
