# Plugin: Gestão de Planos de Saúde

## O que faz
Automatiza guias, TISS e controle de glosas com alertas inteligentes.

## Funcionalidades
- Geração automática de guias TISS ao agendar procedimento
- Verificação de elegibilidade em tempo real
- Controle de prazo de guias (alertas D-5)
- Análise de glosas com sugestão de recurso
- Dashboard de faturamento mensal por plano

## Compliance
- LGPD: dados de saúde tratados como dados sensíveis
- Agente @datum (LGPD) valida todos os fluxos de dados
- Conformidade com resoluções ANS

## Integração
- Sistemas TISS dos planos de saúde
- SADT digital
- n8n para automação de fluxos

## Alertas Automáticos
- Guia próxima do vencimento
- Glosa identificada → sugestão de recurso gerada automaticamente
- Produção abaixo da meta → alerta para gestão
