# Plugin: Captação de Novos Pacientes

## O que faz
Funil de captação via Google e Meta com qualificação automática por especialidade.

## Funil de Captação
`
Anúncio (Google/Meta) segmentado por especialidade + região
  → Lead Form integrado (sem sair da plataforma)
  → Qualificação automática via WhatsApp (plano, urgência, localização)
  → Score de qualificação calculado
  → Agendamento imediato se score >= 70
  → Nutrir com conteúdo educativo se score < 70
`

## Copy dos Anúncios
- Gerado por @delta + @vivid (squad-copy)
- Aborda dor específica da especialidade
- CTA: "Agende em 2 minutos via WhatsApp"

## Integrações
- Meta Ads Lead API
- Google Ads API
- WhatsApp Cloud API
- n8n

## MCP
Nenhum necessário
